This directory consists of the following subdirectories:

    * iostreams contains source files for the non-ObjectStore implementation
      of the reservations application, as decribed in "Writing ObjectStore
      Applications"

    * basic_os contains source files for the basic ObjectStore implementation
      of the reservations application, as decribed in "Writing ObjectStore
      Applications"

    * coll_os contains source files for the collections implementation
      of the reservations application, as decribed in "Writing ObjectStore
      Applications"
