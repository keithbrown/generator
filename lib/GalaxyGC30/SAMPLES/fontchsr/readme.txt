This directory contains two demos showing the use of the font chooser:

	fontchsr     Shows setup and use of standard, no-frills
		     font chooser.

	fontchs2     Shows setup and use of subclassed file chooser
		     to filter on font families, family faces, and
		     font sizes. Also shows how to prevent user 
		     customization to the global font list.

Files:

	fontchsr
		fontchsr.c      C source
		fontchsr.vr     Resource file

	fontchs2
		fontchs2.c      C source
		fontchs2.vr     Resource file
