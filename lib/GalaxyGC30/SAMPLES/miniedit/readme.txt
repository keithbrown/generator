

miniedit - a small and simple word processor
--------------------------------------------

This miniedit sample consists of a very basic and simple word
processor.  Visual components include:

	text window imported from vdialog and vtextview managers
	font chooser imported from vfontchsr manager.

The text window displays all words you typed and/or selected
The font chooser controls the font styles on
   (1) the current I-beam cursor position in text window
   (2) the selected part you made in text window

You may run the miniedit application as:

    % miniedit

You may exit the miniedit application by using system menu.
