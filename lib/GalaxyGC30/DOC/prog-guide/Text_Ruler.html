Text Ruler



A text ruler is a dialog item (vtextruler) that is typically included in applications to allow end users to manipulate text data objects displayed by a text view or text item. A text ruler contains controls that perform operations on an associated text data object and text selection. These operations include setting tabs, setting justification, setting line spacing, adjusting the left, right and first line margins, and adjusting all attributes of a vtextRuler (refer to the Text Manager). The text ruler always reflects the state of the associated selection, and reflects those attributes that have indeterminate state. For example, the justification controls are grayed out if the selection includes lines of text that have different justification values.
