Installation



The Installation class can be used to find the resource files needed by an application regardless of where the application has been installed. You do not need to call the Installation class directly if your application uses the Application class to access its resource files (this is recommended). However, if you need more than one application-specific resource file, or you do not want to name your resource file based on your application executable name, you need to use the Installation class.
