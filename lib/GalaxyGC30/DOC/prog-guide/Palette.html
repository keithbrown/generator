Palette



A palette is a dialog item that manages a rectangular array of images or icons, as shown in Figure 67. The palette contains a set of palette items. The palette items are indexed starting with the top left item, as illustrated in Figure 68. Each palette item usually contains an icon or image representing an object or action, and a selection notification function that gives notification when the palette item is selected. The palette can contain only one selection at a time. When a palette item is selected, the user application is notified to take the proper action.
