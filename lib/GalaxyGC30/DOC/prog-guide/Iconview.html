Iconview



An icon view is an special version of a domain item, that allows named icons to be placed and manipulated in an icon view domain. Icons can share the same image, or they can all have individual images.

Since an icon view is a sub-class of a domain item, all domain item and domain view operations apply to an icon view. You can also obtain the domain for the icon view and use any domain operation on the domain as well. For example, you may want to add a rectangle, lines, or some other type of object to the icon view. You can create and add your own objects to the icon view's domain, without affecting the behavior of the icon view.

Key Functionality

To snap icons to the grid while dragging, call:

    vdomainviewSetSnapToGrid(viconviewGetDomainView(iconview), vTRUE);
This makes icon view extremely flexible. Icon views can also be created with the Visual Resource Builder, and can be stored in and loaded from resource files.
