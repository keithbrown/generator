Programmer's Guide



This online document is the Galaxy Programmer's Guide; it discusses each of the Galaxy managers. It is intended for software engineers using Galaxy to design or write application programs.

This manual assumes a basic understanding of the C programming language, as well as object-oriented design concepts and event-driven programming.
