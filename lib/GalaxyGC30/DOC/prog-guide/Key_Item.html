Key Item



A key item is a dialog item subclass that exists simply to record keystrokes. A key item will display the last entered or set keystroke, including all modifiers. Use key items to request from the user a keystroke. Typically, key items are used to obtain a keystroke that will invoke an action (key binding). Key items can also be created with the Visual Resource Builder, and can be stored in and loaded from resource files.
