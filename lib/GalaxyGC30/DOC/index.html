The following help is available:
  • Getting Started
    Helpful links and suggested starting points for the new user.

  • Help on Help
    Information on accessing different parts of the Visual Resource Builder Help system.

  • Galaxy Tutorials
    Contains links to several tutorials, helping you to quickly get up and going with Galaxy.

  • Galaxy Samples
    Information about building and running the Galaxy samples, along with categorical and alphabetical lists of the various samples.

  • Programmer's Guide
    Concepts and ideas relating to individual Galaxy classes. Includes numerous code examples.

  • Reference Manual
    The API reference points, giving details on the methods, variables, and constructors for the Galaxy classes and interfaces.

  • Visual Resource Builder
    The Visual Resource Builder is a development tool for creating the user interface portion of applications. VRB help is divided into two portions:

    • Visual Resource Builder Editors
      Information on the various Editors available in Visual Resource Builder, including details on how to access and use the many editors effectively when building your application.

    • Object Palette
      Provides documentation for the Object Palette. Each page or object of the palette is described, explaining how to use the different components to build your application.

  • Other Tools
    Covers the various other tools provided with Galaxy, including the resource tools.
