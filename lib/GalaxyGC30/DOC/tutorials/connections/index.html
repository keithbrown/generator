Connections Tutorial

In this tutorial, you learn how to use springs and struts, interactive geometry management tools in the Visual Resource Builder. Springs and struts are "connections" between objects that control the position and size of objects in your application's user interface. All of the connections used in a particular dialog or container are called a "spring solution."

The Visual Resource Builder provides interactive editors to specify and test spring solutions. The spring solutions are automatically saved in your project files, and can be modified as you make additions or changes to your application. A dialog's spring solution is independent of the application code, allowing you to specify geometry management in the Visual Resource Builder without recompiling or modifying the underlying code.
