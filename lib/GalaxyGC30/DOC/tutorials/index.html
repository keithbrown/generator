Tutorials

The following lessons are available:

  • Overview of Galaxy
    This tutorial demonstrates basic usage of Galaxy, touching on many concepts and classes. If you are new to Galaxy, this is the place to start learning.

  • Using Connections
    Shows how to use springs and struts--Galaxy's interactive geometry management tools. Springs and struts are "connections" between objects that control the position and size of objects in your application's user interface.
