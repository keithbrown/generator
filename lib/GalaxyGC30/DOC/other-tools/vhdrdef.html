vhdrdef



Usage

 Command line: vhdrdef [-remove] inputfile ...

vhdrdef converts include file specifications in source code files to macros defined in vport.h. The conversion is done in place, overwriting the contents of the input files. This utility updates include file specifications to the new format introduced with Release 2.0 of Galaxy.

Previous releases of Galaxy included header files using standard C syntax, as shown in Example 1.

Example 1. Old format include file definitions 

    #include <vstdlib.h>
#include <vresource.h>
#include <vapplication.h>
The new format for including header files maintains compatibility for header files across all platforms. Example 2 shows the new style for including the Galaxy header files listed in Example 1.

Example 2. New format include file definitions 

    #include <vport.h>
#include vstdlibHEADER
#include vresourceHEADER
#include vapplicationHEADER

Options
-remove
Converts new format specifications to the old format.

inputfile ...
List of header files to convert.

    NOTE: After conversion to the new format, the statement #include <vport.h> must be added before any statement that includes Galaxy header files.

    NOTE: If vport.h was present in the file before running vhdrdef, check to see if it was (incorrectly) converted to the new style.
