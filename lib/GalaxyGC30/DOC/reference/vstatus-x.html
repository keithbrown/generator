Types

vstatus

        typedef ... vstatus;
A generic result code that fits within a 32-bit quantity. vstatus variables are returned from some Galaxy functions to indicate the success or failure of a function call. Compare vstatus variables with constants defined in the vstatus header file, such as vstatusSUCCESS or vstatusTIMED_OUT.

The specific meaning of each of the constants defined by vstatus is determined by the function from which it was returned.

vstatusINCLUDED

        vstatusINCLUDED
Defined when the header file vstatus.h is included. Use this constant to protect against multiple inclusion of the vstatus header file. This constant is intended to be used in this way when including header files: 
#ifndef  vstatusINCLUDED
#include vstatusHEADER
#endif
