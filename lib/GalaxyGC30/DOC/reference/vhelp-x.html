        vhelpWindow * vhelpCreateWindow();
(Deprecated) Creates a new help window of the current default help window class. The default help window class is controlled with vhelpSetDefaultWindowClass. vhelpSwitchToNestedContext

        void vhelpSwitchToNestedContext(
                vhelpWindow * window,
                const vchar ** arrayContext,
                int count,
                const vrect * rectPlace);
(Deprecated) vhelpSwitchWindowToNestedContext is now the preferred call.

The most specific available context, if any, that matches the request is shown in the help window window. arrayContext is an array of nArray strings that are used to make up the context. The actual context name searched is the list of strings appended together; the strings are concatenated together and tried, they are then backed off until a match is found. The strings are used in reverse order. For example:

  • Item
  • Group, and
  • dialog

look for the contexts:

  • dialogGroupItem
  • dialogGroup, then just
  • dialog.

If rectPlace is not NULL, an attempt is made to place the help window next to that rectangle. This is implemented by sending vhelpSWITCH_NESTED_CONTEXT to the help window.
