Types [4] Functions [17] Messages [3] Constants [3] Macros [1]

Types

vslider

        typedef ... vslider;
Contains the information needed to represent a slider control. A slider is a dialog item which lets the user drag over a linear range of values.

vsliderClass

        typedef ... vsliderClass;
Contains method and variables bindings for slider objects of type vcontrol. Classes declared with this type contain the same set of methods and variables as the default slider class, but the behavior of methods and the contents of variables may vary. This class type is derived from vcontrolClass.

vsliderScrollBar

        typedef ... vsliderScrollBar;
Contains the information needed to represent a sliderScrollbar control. A sliderScrollbar is essentially a slider control with increment arrows that allow the scroll bar value to be changed by fixed amounts.

vsliderScrollBarClass

        typedef ... vsliderScrollBarClass;
Contains method and variables bindings for objects of type vsliderScrollBar. Classes declared with this type contain the same set of methods and variables as the slider control class, but the behavior of methods and the contents of variables may vary. This class type is derived from vsliderClass.

vsliderCalcContent

        void vsliderCalcContent(
                vslider * slider,
                vrect * content);
Calculates the content rectangle of the slider slider and stores it in content. The content rectangle is the region over which the thumb of the slider can slide. This function is implemented by sending the message vsliderCALC_CONTENT to slider.

vsliderCalcThumb

        int vsliderCalcThumb(
                vslider * slider,
                int thin,
                int thick);
Returns the length of the thumb of the slider slider according to the size of the content rectangle. thin is the smaller dimension (width or height) of the content rectangle, and thick is the larger dimension. This function is implemented by sending the message vsliderCALC_THUMB to slider.

vsliderCreate

        vslider * vsliderCreate();
Creates a new slider of the default slider class and returns a pointer to its contents. Storage for the slider is allocated by the Control class. The Control class reclaims the storage occupied by the slider when it is destroyed with vcontrolDestroy. The new slider is created with a set of default attributes, as determined by the default slider class. The default attributes of the default slider class are documented in the ``Control Manager'' chapter of the Galaxy Application Environment Programmer's Guide. This function essentially allocates space for a new slider and calls vsliderInit on it.

vsliderCreateOfClass

Creates a new slider of class clas and returns a pointer to its contents. Storage for the slider is allocated by the Control class. The Control class reclaims the storage occupied by the slider when it is destroyed with vcontrolDestroy. The new slider is created with a set of default attributes, determined by the class. The default attributes of the default slider class are documented in the ``Control Manager'' chapter of the Galaxy Application Environment Programmer's Guide. This function essentially allocates space for a new slider and calls vsliderInitOfClass on it.

vsliderCreateScrollBar

        vsliderScrollBar * vsliderCreateScrollBar();
Creates a new scroll bar of the default scroll bar class and returns a pointer to its contents. Storage for the scroll bar is allocated by the Control class. The Control class reclaims the storage occupied by the scroll bar when it is destroyed with vcontrolDestroy. The new scroll bar is created with a set of default attributes, as determined by the default scroll bar class. The default attributes of the default scroll bar class are documented in the ``Control Manager'' chapter of the Galaxy Application Environment Programmer's Guide.

The default scrollbar class provides a special feature under OPEN LOOK and Windows. Under these look-and-feels the scrollbar is automatically disabled when an entire view is visible according to scrollbar attributes. That is, when the view attribute is less than the maximum minus the minimum attribute. Refer to vsliderSetView for more information. This function essentially allocates space for a new scroll bar and calls vsliderInitScrollBar on it.

vsliderGetDefaultClass

        vsliderClass * vsliderGetDefaultClass();
Returns the class used to create slider controls by the vsliderCreate and vsliderInit functions. This class can be modified to alter the behavior of all slider controls subsequently created by this function. Changes to this class are not visible to any other applications in the same process or sharing the same display server. This function can be called to explicitly register the slider control class with the Class manager. Sliders can be used to select over a linear range of values.

vsliderGetScrollBarClass

        vsliderScrollBarClass * vsliderGetScrollBarClass();
Returns the class used to create scroll bar controls by the vsliderCreateScrollBar and vsliderInitScrollBar functions. This class can be modified to alter the behavior of all scroll bar controls subsequently created by this function. Changes to this class are not visible to any other applications in the same process or sharing the same display server. This function can be called to explicitly register the scroll bar control class with the Class class. A scroll bar can be used to select over a linear range of values.

vsliderInit

        void vsliderInit(
                vslider * slider);
Initializes a slider of the default slider class in the memory addressed by slider. The attributes of the slider are set to their default values, according to the default slider class. The default attributes of the default slider class are documented in the ``Control Manager'' chapter of the Galaxy Application Environment Programmer's Guide. The address can be used in other calls to the Control class as if it were created with vsliderCreate. A call to vcontrolDestroy frees any internal objects allocated by the Control class, but does not free the structure itself. This function simply stores the default slider class in the class field of the slider and sends it the message vobjectINIT.

vsliderSetValueScroll

        void vsliderSetValueScroll(
                vslider * slider,
                int value);
Changes the value of the slider slider and immediately updates the slider's appearance, if open. The slider is not invalidated. This function may result in better interactive performance than vcontrolSetValue, but it should only be called when interactive drawing is necessary since it results in excessive redrawing if used carelessly. This function is implemented by sending the message vsliderSET_VALUE_SCROLL to slider.

Messages

vsliderCALC_CONTENT

        <vsliderCALC_CONTENT>
Calculates the content rectangle of the slider slider and stores it in content. The content rectangle is the region over which the thumb of the slider can slide. The function entry point vsliderCalcContent is implemented by sending this message to the slider.

The default method for the default slider class calculates the content rectangle of the control based on its bounding rectangle. Subclasses that override this method should first call vclassSendSuper to calculate the slider content rectangle, then include any subclass-specific contents.

vsliderCALC_THUMB

        <vsliderCALC_THUMB>
Returns the length of the thumb of the slider slider according to the size of the content rectangle. thin is the smaller dimension (width or height) of the content rectangle, and thick is the larger dimension. The function entry point vsliderCalcThumb is implemented by sending this message to the slider.

The default method for the default slider class calculates the thumb size based on the current look-and-feel. Subclasses that override this method should first call vclassSendSuper to calculate the slider thumb size, then include any subclass-specific size.

vsliderSET_VALUE_SCROLL

        <vsliderSET_VALUE_SCROLL>
Changes the value of the slider slider and immediately updates the slider's appearance, if open. The slider is not invalidated. This message may result in better interactive performance than vcontrolSetValue, but it should only be sent when interactive drawing is necessary since it results it excessive redrawing if used carelessly. The function entry point vsliderSetValueScroll is implemented by sending this message to the slider.

The default method for the default slider class moves the slider's indicator rectangle by scrolling its contents. Subclasses that override this method should first call vclassSendSuper to change the slider value, then update any subclass-specific state.

Constants

vsliderINCLUDED

Defined when the header file vslider.h is included.

vsliderNEXT_ID_ATTRIBUTE

        vsliderNEXT_ID_ATTRIBUTE
Denotes the next available attribute ID for subclasses of the default slider class. This constant is provided so that attribute IDs defined by subclasses do not conflict with those defined by the Control Manager. Only attributes defined by overriding the methods vobjectSET_ID_ATTRIBUTE and vobjectGET_ID_ATTRIBUTE need be concerned with the attribute IDs of their superclasses. Subclasses of vsliderClass should begin their attribute IDs with this constant and increment by one for each attribute. Subclasses should also provide their own NEXT_ID_ATTRIBUTE constant for use by their subclasses. This constant has integral type.

vsliderNEXT_SELECT

        vsliderNEXT_SELECT
Denotes the first command selector available to subclasses of the default slider class. This constant is provided so that selector values defined by subclasses do not conflict with those defined by the Control class. Only commands defined by overriding the methods vobjectISSUE and vobjectQUERY need be concerned with the selector values of their superclasses. Subclasses of vsliderClass should begin their command selectors with this constant and increment by one for each new command. Subclasses should also provide their own NEXT_SELECT constant for use by their subclasses. This constant has integral type.

Macros

vsliderCLASS

        #define vsliderCLASS(...) ...
Contains the method and variable declarations for slider classes. This macro expands to the contents of vsliderClass. SUPER is the type identifier of the class being declared; for example, vsliderClass. This type is used by the class's subclasses to refer to it. SLIDER is the type identifier of instances of the class being declared; for example, vcontrol. NOTIFY is the type identifier of the instance notification function; for example, vcontrolNoteProc.

This macro should be used by slider subclasses to declare the contents of the slider class. This macro should come first in the subclass structure declaration.
