Types [2] Functions [34] Messages [7] Constants [16] Macros [1]

Types

vtextruler

        typedef ... vtextruler;
Contains the information needed to represent a single text ruler and its contents. A text ruler is a dialog item that visually represents the formatting characteristics of a selection of text.

vtextrulerClass

        typedef ... vtextrulerClass;
Describes the behavior of a class of text rulers. Text ruler classes are referenced by a pointer to this type.

Functions

vtextrulerClone

        vtextruler * vtextrulerClone(
                vtextruler * ruler);
Creates a copy of the text ruler ruler and returns a pointer to its contents. This function is essentially a combination of vtextrulerCreate and vtextrulerCopy.

vtextrulerCopy

        void vtextrulerCopy(
                vtextruler * ruler,
                vtextruler * target);
Copies the information in textRuler into target, so that the two rulers represent equivalent objects. target must be created with vtextrulerCreate or an equivalent function.

vtextrulerCopyInit

        void vtextrulerCopyInit(
                vtextruler * ruler,
                vtextruler * target);
Copies the information in text ruler into target, so that the memory addressed by target is a text ruler equivalent to ruler. target should be a pointer to uninitialized memory; the text ruler initializes it as part of the copy.

vtextrulerCreate

        vtextruler * vtextrulerCreate();
Creates a new text ruler of the default class and returns its pointer.The text ruler is created with a set of default attributes. If a different set of attributes is desired, they must be set explicitly.

vtextrulerCreateOfClass

        vtextruler * vtextrulerCreateOfClass(
                vtextrulerClass * clas);
Creates a new text ruler of class clas and returns a pointer to it. The new text ruler is created with a set of default attributes determined by the class. If a different set of attributes is desired, they must be set explicitly.

vtextrulerDestroy

        void vtextrulerDestroy(
                vtextruler * ruler);
Destroys the text ruler represented by ruler.

vtextrulerGetClass

        vtextrulerClass * vtextrulerGetClass(

Returns the text ruler class for the text ruler ruler.

vtextrulerGetData

        const void * vtextrulerGetData(
                vtextruler * ruler);
Returns the generic data pointer attached to the text ruler textRuler, which can be NULL.

vtextrulerGetDefaultClass

        vtextrulerClass * vtextrulerGetDefaultClass();
Returns the class used to create rulers by the vtextrulerCreate and vtextrulerInit functions. This class can be modified to alter the behavior of rulers subsequently created by these functions.

vtextrulerGetDefaultObserverClass

        vtextObserverClass * vtextrulerGetDefaultObserverClass();
Returns the class that contains the default observer class methods for text rulers. Text rulers register this observer class with the text data object associated with the ruler (with vtextrulerSetTextData) to receive notification of text data selection changes. You should set the methods of this class to override or enhance observer methods. Refer to ``Observers'' in the ``Text Manager'' chapter of the Galaxy Application Environment Programmer's Guide for more information.

vtextrulerGetDefaultRuler

        vtextRuler * vtextrulerGetDefaultRuler(
                vtextruler * ruler);
Returns a pointer to the default ruler for the text ruler ruler. The default ruler is displayed and modified by the text ruler if the Use_Default attribute is set, or if there is not a text data object or selection associated with the text ruler.

vtextrulerGetItem

        vdialogItem * vtextrulerGetItem(

Returns the dialog item to which the text ruler ruler belongs.

vtextrulerGetLeftMargin

        short vtextrulerGetLeftMargin(

Returns the left margin for the text ruler ruler. The margin specifies where the text ruler should consider the left edge of the document.

vtextrulerGetNotify

Returns the notification procedure for the text ruler ruler.

vtextrulerGetObserverClass

        vtextObserverClass * vtextrulerGetObserverClass(
                vtextruler * ruler);
Returns the observer class for the text ruler ruler that contains the methods that the text data object calls when the text data changes.

vtextrulerGetScrollOffset

        int vtextrulerGetScrollOffset(
                vtextruler * ruler);
Returns the scroll offset for the text ruler ruler. The offset is the (absolute) number of pixels the ruler is shifted over.

vtextrulerGetSelection

Returns the selection associated with the text ruler ruler. It returns NULL if there is no selection associated with the text ruler.

vtextrulerGetTextData

        vtext * vtextrulerGetTextData(
                vtextruler * ruler);
Returns the text data object associated with the text ruler ruler. NULL is returned if there is no text data object associated with the text ruler.

vtextrulerGetUseDefault

        int vtextrulerGetUseDefault(
                vtextruler * ruler);
Returns TRUE if the use default attribute is set for the text ruler ruler, FALSE otherwise.

vtextrulerInit

        void vtextrulerInit(
                vtextruler * ruler);
Makes the text ruler addressed by textRuler a valid text ruler of the default class in the same manner as vtextrulerInitOfClass.

vtextrulerInitOfClass

        void vtextrulerInitOfClass(
                vtextruler * ruler,
                vtextrulerClass * clas);
Makes the text ruler addressed by ruler a valid text ruler of class clas in the same manner as vtextrulerCreateOfClass , except that storage is not allocated for the text ruler. If the text ruler is passed to vtextrulerDestroy, any internally allocated storage is freed, but the text ruler itself is not freed. The storage addressed by ruler need not be dynamically allocated.

vtextrulerLoad

        vtextruler * vtextrulerLoad(
                vresource resource);
Reads a text ruler from the resource resource and returns a pointer to its contents. The contents are allocated by the text ruler. An exception is generated if the resource has not been constructed properly.

vtextrulerLoadInit

        void vtextrulerLoadInit(
                vtextruler * ruler,
                vresource resource);
Reads a text ruler from the resource resource and stores it in the structure addressed by textRuler, which should be uninitialized. An exception is generated if the resource has not been constructed properly.

vtextrulerSetData

        void vtextrulerSetData(
                vtextruler * ruler,
                const void * data);
Sets a text ruler's generic data pointer to data. This can be a pointer to whatever information the client wants to attach to the text ruler; it is not used by the text ruler. data can be NULL.

vtextrulerSetDefaultRuler

        void vtextrulerSetDefaultRuler(
                vtextruler * ruler,
                vtextRuler * defaultRuler);
Sets the default ruler to defaultRuler for the text ruler ruler. The default ruler is displayed and changed by the text ruler if the text ruler's text data object or selection is NULL, or if the text ruler attribute use default is set.

vtextrulerSetLeftMargin

        void vtextrulerSetLeftMargin(
                vtextruler * ruler,
                int leftMargin);
Sets the left margin for the text ruler ruler. The margin specifies where the text ruler should consider the left edge of the document.

vtextrulerSetNotify

        void vtextrulerSetNotify(
                vtextruler * ruler,
                vtextrulerNoteProc noteProc);
Changes the notification procedure for the text ruler ruler to noteProc. Whenever the text data ruler that the text ruler reflects changes, this procedure is called with the text ruler, and the event that caused the ruler change (this can be NULL, indicating the change was not generated from a event). noteProc can be NULL, in which case no notification is given.

vtextrulerSetObserverClass

        void vtextrulerSetObserverClass(
                vtextruler * ruler,
                vtextObserverClass * observerClass);
Sets the observer class for the text ruler textRuler. Call this routine if you want to install your own observer class. The observer class should be derived from the text ruler default observer class obtained with vtextrulerGetDefaultObserverClass.

vtextrulerSetScrollOffset

        void vtextrulerSetScrollOffset(
                vtextruler * ruler,
                int offset);
Sets the scroll offset for the text ruler ruler.

vtextrulerSetSelection

        void vtextrulerSetSelection(
                vtextruler * ruler,
                vtextSelection * selection);
Associates the selection selection with the text ruler ruler. Usually, you also want to associate a text data object for the text ruler to monitor as well, using vtextrulerSetTextData. With both the text data and selection set, changes to the text ruler affect the selected text in the designated text data object, and likewise changes to the selection result in changes to the text ruler. The text ruler's default ruler is used if either the text data object or selection is not set. Pass NULL for selection to disassociate the selection with the text ruler.

vtextrulerSetTextData

        void vtextrulerSetTextData(
                vtextruler * ruler,
                vtext * text);
Associates the text data object text with the text ruler ruler. Usually, you also want to associate a selection for the text ruler to monitor as well, using vtextrulerSetSelection. With both the text data and selection set, changes to the text ruler affect the selected text in the designated text data object, and likewise changes to the selection result in changes to the text ruler. The text ruler's default ruler is used if either the text data object or selection is not set. Pass NULL for text to disassociate the text data object with the text ruler.

vtextrulerSetUseDefault

        void vtextrulerSetUseDefault(
                vtextruler * ruler,
                int useDefault);
Sets the Use_Default attribute for the text ruler ruler. If useDefault is TRUE, then the text ruler's default ruler is displayed and modified by the text ruler, otherwise, the associated text data object and selection is used by the text ruler. If there is not a text data object or selection associated with the text ruler, the default ruler is used regardless of the Use_Default attribute.

vtextrulerStartup

        void vtextrulerStartup();
Initializes the manager.

vtextrulerStore

        void vtextrulerStore(
                vtextruler * ruler,
                vresource resource);
Writes the contents of the text ruler ruler into the resource resource. The text ruler can be recovered from the resource at a later time (perhaps by another application) with vtextrulerLoad or vtextrulerLoadInit. The resource should have type vresourceDICT.
