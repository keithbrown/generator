vdrawrectset

        typedef ... vdrawrectset;
The vdrawrectset class provides API onto generic operations on the set of rectangles.

vdrawrectsetClass

        typedef ... vdrawrectsetClass;
Contains method and variable bindings for objects of type vdrawrectset. This class type is derived from vloadableClass.

vdrawrectsetGetDefaultClass

        vdrawrectsetClass * vdrawrectsetGetDefaultClass();
Returns the class used to create dialogs by the vdrawrectsetCreate and vdrawrectsetInit functions. This class can be modified to alter the behavior of all rectangle sets subsequently created by these functions. Changes to this class are not visible to any other applications in the same process or to those sharing the same display server. This function can be called to explicitly register the default rectangle set class with the Class Manager.

vdrawrectsetCLASS

        #define vdrawrectsetCLASS(...) ...
Contains the method and variable declarations for rectangles set classes. This macro expands to the contents of vdrawrectsetClass. CLASS is the type identifier of the class being declared (for example, vdrawrectsetClass). This type is used by the class's subclasses to refer to it. RECTSET is the type identifier of instances of the class being declared (for example, vdrawrectset). This macro should be used by rectangles set subclasses to declare the contents of the rectangles set class. This macro should come first in the subclass structure declaration.
