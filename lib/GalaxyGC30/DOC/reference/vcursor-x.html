Types

vcursor

        typedef ... vcursor;
Defines a cursor, which is a device-independent representation of a screen cursor. A cursor consists primarily of an image, a mask, and a hot spot.

vcursorClass

        typedef ... vcursorClass;
Contains the message definitions for a class of cursors. This class type is derived from vimageClass.

vcursorDestroy

        void vcursorDestroy(
                vcursor * cursor);
Destroys a previously initialized cursor. Whether you create and/or initialize cursors using vcursorCreate, vcursorInit, or vcursorCopyInit, destroy all cursors when you are finished using them to free up the system resources the cursors may have privately allocated.

vcursorDump

        void vcursorDump(
                vcursor * cursor,
                FILE * stream);
Prints the contents of the cursor cursor on the file stream in user-readable form. This function is intended for debugging and should not be used to store the contents of the image in recoverable form. This function has effect only if debugging is enabled.

Messages

vcursorSET_HOT

        <vcursorSET_HOT>
Sets the hot spot of the cursor cursor to the x,y coordinates as measured from the lower left.

Subclasses that override this method should first call vclassSendSuper to set the new hot spot, then update any subclass-specific state.

Macros

vcursorCLASS

        #define vcursorCLASS(...) ...
Contains the method and variable declarations for notebook classes. This macro expands to the contents of vcursorClass. SUPER is the type identifier of the class being declared. This type is used by the class's subclasses to refer to it. CURSOR is the type identifier of instances of the class being declared.
