Types [3] Functions [24] Messages [8] Constants [8] Macros [1]

Types

vfontchsr

        typedef ... vfontchsr;
A font chooser provides a common interface to the user for choosing fonts. A font chooser can be created and used by any Galaxy application and is the recommended method for requesting a font from the user.

vfontchsrApplyFunc

        typedef ... vfontchsrApplyFunc;
Denotes an apply function that is called when a font has been chosen from the font chooser. font specifies the font that was chosen. The apply function should return TRUE if the font was acceptable, in which case the font chooser closes if appropriate, or FALSE if the font is not acceptable, in which case the font chooser does not close under any circumstances.

vfontchsrClass

        typedef ... vfontchsrClass;
Describes the behavior of a class of font choosers. The font chooser class controls font application. Font chooser classes are referenced by a pointer to this type.

Functions

vfontchsrClose

        void vfontchsrClose(
                vfontchsr * fontChooser);
Closes the font chooser fontChooser so that it is no longer visible. This function has no effect on a font chooser that is already closed.

vfontchsrCreate

        vfontchsr * vfontchsrCreate();
Creates a new font chooser of the default font chooser class and returns a pointer to the new font chooser. (Memory is allocated by the Font Chooser.)

vfontchsrCreateOfClass

        vfontchsr * vfontchsrCreateOfClass(
                vfontchsrClass * clas);
Creates a new font chooser of class clas and returns a pointer to the new font chooser. (Memory is allocated by the Font Chooser.) The new font chooser is created with a set of default attributes determined by the class.

vfontchsrDestroy

        void vfontchsrDestroy(
                vfontchsr * fontChooser);
Destroys the font chooser represented by fontChooser and all of its sub-objects that were allocated by the Font Chooser. This function attempts to free the storage directly occupied by font chooser only if it was allocated by vfontchsrCreate or vfontchsrCreateOfClass .

vfontchsrGetApply

        vfontchsrApplyFunc vfontchsrGetApply(
                vfontchsr * fontChooser);
Returns the apply function for the font chooser fontChooser, or NULL if there is no apply procedure.

vfontchsrGetClass

        vfontchsrClass * vfontchsrGetClass(
                vfontchsr * fontChooser);
Returns the font chooser class of the font chooser chooser.

vfontchsrGetConfirm

        vconfirm * vfontchsrGetConfirm(
                vfontchsr * fontChooser);
Returns the confirmation dialog (vconfirm) upon which the font chooser fontChooser is built. Confirmation Manager operations on this confirm affect the font chooser it came from. All Confimation Manager operations have effect, but some may interfere with the font chooser's behavior.

vfontchsrGetData

        void * vfontchsrGetData(
                vfontchsr * fontChooser);
Returns the generic data pointer attached to the font chooser fontChooser, which can be NULL.

vfontchsrGetDefaultClass

        vfontchsrClass * vfontchsrGetDefaultClass();
Returns the class used to create a font chooser by the vfontchsrCreate function. This class can be modified to alter the behavior of all font choosers subsequently created by these functions.

vfontchsrGetDialog

        vdialog * vfontchsrGetDialog(
                vfontchsr * fontChooser);
Returns the dialog upon which the font chooser fontChooser is built. Dialog Manager operations on this dialog affect the font chooser it came from. All Dialog Manager operations have effect, but some may interfere with the font chooser's behavior.

vfontchsrGetFilterOptions

        unsigned vfontchsrGetFilterOptions(
                vfontchsr * fontChooser);
Returns the filter options for a font chooser. The returned options will be a combination of the bit constants with the prefix vfontchsrFILTER_OPTION.

vfontchsrGetFont

        vfont * vfontchsrGetFont(
                vfontchsr * fontChooser);
Returns the current font for the font chooser fontChooser.

vfontchsrGetShowUpright

        int vfontchsrGetShowUpright(
                vfontchsr * fontChooser);
Returns whether only upright fonts or all fonts are displayed in the font chooser fontChooser.

vfontchsrOpen

        void vfontchsrOpen(
                vfontchsr * fontChooser);
Opens the font chooser fontChooser so that it is visible at its current position and size on the screen. This function has no effect if fontChooser is already open.

vfontchsrPlace

        void vfontchsrPlace(
                vfontchsr * fontChooser,
                vdialog * base,
                int modeX,
                int modeY);
Positions the font chooser relative to the given dialog, base. If base is NULL, this function positions the font chooser relative to bounds of the display. modeX and modeY specify the relation of the font chooser to the base dialog on each coordinate axis; appropriate flags are defined by the Plane Geometry Manager function vrectPlace.

vfontchsrProcess

        const vname * vfontchsrProcess(
                vfontchsr * fontChooser);
Opens a blocking font chooser fontChooser and waits until a confirmation button (OK or Cancel) is activated or the user closes the font chooser's window. It is important to note that the font chooser's apply function is called if the user selects the OK button. The font chooser's window is closed before vfontchsrProcess returns.

vfontchsrRescan

        void vfontchsrRescan(
                vfontchsr * fontChooser);
Modify the font chooser lists to reflect changes caused by new font information filtering. This function is called automatically each time the filter options are changed for a font chooser.

vfontchsrSetApply

        void vfontchsrSetApply(
                vfontchsr * fontChooser,
                vfontchsrApplyFunc applyFunc);
Changes the apply function for the font chooser fontChooser to applyFunc. Whenever a font is chosen from the font chooser, the apply function is called, passing the font that was chosen. applyFunc can be NULL, in which case no apply notification is given.

vfontchsrSetData

        void vfontchsrSetData(
                vfontchsr * fontChooser,
                void * data);
Sets a font chooser's generic data pointer to data. This can be a pointer to whatever information the client wants to attach to the font chooser---it is not used by the Font Chooser. data can be NULL.

vfontchsrSetFilterOptions

        void vfontchsrSetFilterOptions(
                vfontchsr * fontChooser,
                unsigned options);
Sets the font information filter options for a font chooser. The filter options can be any combination of the bit constants with the prefix vfontchsrFILTER_OPTION.

vfontchsrSetFont

        void vfontchsrSetFont(
                vfontchsr * fontChooser,
                vfont * currentFont);
Sets the current font to font for the font chooser fontChooser. The current font is the font that is highlighted (if the font chooser is open) or will be highlighted (if the font chooser is closed) in the font chooser.

vfontchsrSetLeader

        void vfontchsrSetLeader(
                vfontchsr * fontChooser,
                vwindow * leader);
Sets the leader of the font chooser fontChooser to the window leader. leader can be NULL in which case the root window is used as the chooser's leader. Refer to vwindowGetRoot for a description of the root window.

vfontchsrSetShowUpright

        void vfontchsrSetShowUpright(
                vfontchsr * fontChooser,
                int showUpright);
Determines whether the font chooser fontChooser shows only upright fonts or all fonts. If showUpright is TRUE, then only fonts that are upright are displayed by the font chooser; otherwise, all available fonts are displayed.

vfontchsrStartup

        void vfontchsrStartup();
Initializes the Galaxy Font Chooser module. You must do this before creating any font choosers.

Messages

vfontchsrFILTER

        <vfontchsrFILTER>
Returns TRUE if the font font can be included in the font choosers's lists (family, face, and size). This method is called when the font chooser fontChooser opens for the first time, and when the user changes either the selected family or the selected face.

vfontchsrFILTER_FACE

        <vfontchsrFILTER_FACE>
Returns TRUE if the font face face can be included in the font choosers's face list. This method may be called when the font chooser fontChooser opens for the first time, when the user changes the selected family, or when the font chooser's filter options are changed.

vfontchsrFILTER_FAMILY

        <vfontchsrFILTER_FAMILY>
Returns TRUE if the font family family can be included in the font choosers's family list. This method may be called when the font chooser fontChooser opens for the first time, or when the font chooser's filter options are changed.

vfontchsrFILTER_RESCAN

        <vfontchsrFILTER_RESCAN>
Modify the font chooser lists to reflect changes caused by new font information filtering. This method is called automatically each time the filter options are changed for a font chooser.

vfontchsrHOOK_APPLY_FONT

        <vfontchsrHOOK_APPLY_FONT>
Calls the apply function, if set, passing the font that was chosen.

Overridng the HookApplyFont member function is equivalent to providing a hook apply function via SetHookApplyFontFunc. You must do one of these things to provide application-specific action when the user chooses a font from the chooser.

vfontchsrSET_FILTER_OPTIONS

        <vfontchsrSET_FILTER_OPTIONS>
Sets the filter options for a font chooser. The filter options can be any combination of the bit constants with the prefix vfontchsrFILTER_OPTION.

vfontchsrSET_FONT

        <vfontchsrSET_FONT>
Sets the current font to font for the font chooser fontChooser. The current font is the font that is highlighted (if the font chooser is open) or will be highlighted (if the font chooser is closed) in the font chooser.

vfontchsrSET_SHOW_UPRIGHT

        <vfontchsrSET_SHOW_UPRIGHT>
Determines whether the font chooser fontChooser shows only upright fonts or all fonts. If showUpright is TRUE, then only fonts that are upright are displayed by the font chooser; otherwise, all available fonts are displayed.

Macros

vfontchsrCLASS

        #define vfontchsrCLASS(...) ...
Contains the method and variable declarations for font chooser classes. This macro expands to the contents of vfontchsrClass. SUPER is the type identifier of the class being declared; for example, vfontchsrClass. This type is used by the class's subclasses to refer to it. CONFIRM is the type identifier of instances of the class being declared; for example, vfondchsr. NOTIFY is the type identifier of the instance notification function; for example, vdialogItemNoteProc. This macro should be used by font chooser subclasses to declare the contents of the font chooser class. This macro should come first in the subclass structure declaration.
