Types

vhelpgalaxy

        typedef ... vhelpgalaxy;
Unlike other subclasses of vhelphypertext, this display does not use a vwebbrowse instance. Instead, it makes requests to vhelpsrv to display help data. Also, vhelpgalaxy does not operate using URL's, but instead uses help context strings (usually derived from vobjectBuildHelpContext). The display expects the help data of the help request to be an instance of vhelpref. It will pass the context string in the vhelpref to vhelpsrv, which will attempt to resolve the context to an entry in a help document. See the Tools manual for additional information on using the Galaxy hypertext help system.

vhelpgalaxyClass

        typedef ... vhelpgalaxyClass;
Contains message and variable bindings for objects of type vhelpgalaxy . Classes declared with this type contain the same set of methods and variables as the Galaxy hypertext help display class, but the behavior of methods and the contents of variables may vary. This class type is derived from vhelphypertext.

Constants

vhelpgalaxyINCLUDED

        vhelpgalaxyINCLUDED
Use this constant to protect against multiple inclusion of the vhelpgalaxy header file. This constant is intended to be used in this way when including header files: 
#ifndef  vhelpgalaxyINCLUDED
#include vhelpgalaxyHEADER
#endif

Macros

vhelpgalaxyCLASS

        #define vhelpgalaxyCLASS(...) ...
This macro contains the method declarations for the helpgalaxy class, this macro expands to the contexts of vhelpgalaxyClass. SUPER<\SAMP> is the type identifier of the class being declared. This type is used by the class subclasses to refer to it. HELPDIS is the type identifier of the instances of the class being declared. This macro should be used by helpgalaxy subclasses to declare the contents of the helpgalaxy class. This macro should come first in the subclass structure declaration.
