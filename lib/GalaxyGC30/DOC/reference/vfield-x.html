vfieldClass

        typedef ... vfieldClass;
Fields implement an observable piece of data. The data can be accessed using an interface class that implements the accessors for the particular datatype. The base class does not actually have a value and therefore does not implement any of these interfaces. It does however, implement, attributes common to all field classes such as title, tag and format.

        void vfieldLock(
                vfield * field);
A locked field does not announce changes when it's set methods are called. Instead, the announcements are merged together into one aggregate announcement made when vfield is called.

Note that you can make multiple calls to Lock. The field will not be unlocked until the outermost call to vfield is made.
