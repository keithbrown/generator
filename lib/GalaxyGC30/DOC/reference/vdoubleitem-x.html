vdoubleitem

        typedef ... vdoubleitem;
A vdoubleitem is a vtypeditem derivative that represents a double value. The primary means by which DoubleItem carries out it mission is by the use of a vformatDouble object.

By default the item is associated with a locale-specific global vformat object but other, non-shared, instances may be set by SetFormat.

vdoubleitemClass

        typedef ... vdoubleitemClass;
Contains method and variable bindings for objects of type vdoubleitem. Classes declared with this type contain the same set of methods and variables as the default doubleitem class, but the behavior of methods and the contents of variables may vary. This class type is derived from vtypeditemClass.

vdoubleitemGetIncrement

        double vdoubleitemGetIncrement(
                vdoubleitem * item);
DoubleItem's parent class, vtypeditem, supports the notion of incrementable type-specific input items. The vtypeditemIncrement method is called in response to user-initiated actions (like keystrokes) and its lone parameter, of type vtypeditemDirection, specifies whether the item's value is being lowered or raised. The absolute value of the increment can be obtained from this method.
