Types

vfixed

        typedef ... vfixed;
Denotes a fixed-point number. A fixed-point number is a number with a fractional component that has a constant exponent. Fixed-point numbers are based on an integral type, but their exact representation is not visible to clients of the Fixed-Point Library. Fixed-point numbers can be added and subtracted with integer operators; function entry points are provided for other mathematical operations.

Functions

vfixedAbs

        vfixed vfixedAbs(
                vfixed x);
Returns the absolute value of the fixed-point number x.

vfixedCeil

        vfixed vfixedCeil(
                vfixed x);
Returns the ceiling of the fixed-point number x. The ceiling of x is the least integer that is greater than or equal to x. For example, if x = 5.5, the ceiling is 6; if x = -5.5, the ceiling is -5.

vfixedDiv

        vfixed vfixedDiv(
                vfixed x,
                vfixed y);
Divides the fixed-point number x by y and returns the result.

vfixedDivQuick

        vfixed vfixedDivQuick(
                vfixed x,
                vfixed y);
Divides the fixed-point number x by y and returns the result. vfixedDivQuick divides more quickly than vfixedDiv at the expense of accuracy; it is useful for testing boundary conditions but does not calculate precise results.

vfixedFloor

        vfixed vfixedFloor(
                vfixed x);
Returns the floor of the fixed-point number x. The floor of x is the greatest integer that is less than or equal to x. For example, if x = 4.5, the floor is 4; if x = -4.5, the floor is -5.

vfixedFromFloat

        vfixed vfixedFromFloat(
                double x);
Returns the fixed-point equivalent of the floating-point number x. This function returns vfixedONE for 1.0 and vfixedZERO for 0.0.

vfixedFromInt

        vfixed vfixedFromInt(
                int n);
Returns the fixed-point equivalent of the integer n. This function returns vfixedONE for 1 and vfixedZERO for 0.

vfixedMul

        vfixed vfixedMul(
                vfixed x,
                vfixed y);
Multiplies the fixed-point numbers x and y and returns the result.

vfixedMulQuick

        vfixed vfixedMulQuick(
                vfixed x,
                vfixed y);
Multiplies the fixed-point numbers x and y and returns the result. vfixedMulQuick multiplies more quickly than vfixedMul at the expense of accuracy; it is useful for testing boundary conditions but does not calculate precise results.

vfixedRound

        vfixed vfixedRound(
                vfixed x);
Rounds the fractional component of the fixed-point number x. Numbers are rounded to the nearest integer; with .5 rounded up. For example, 1.2 rounds to 1; 1.8 rounds to 2; and 1.5 rounds to 2.

vfixedToFloat

        double vfixedToFloat(
                vfixed x);
Returns the floating-point equivalent of the fixed-point number x.

vfixedToInt

        int vfixedToInt(
                vfixed x);
Returns the integer equivalent of the fixed-point number x. The fractional part of the fixed-point number is truncated.

Constants

vfixedEPSILON

        vfixedEPSILON
The smallest positive fixed-point value such that the values of vfixedONE and vfixedEPSILON when added together do not equal vfixedONE. This constant is of type vfixed.

vfixedFRAC_BITS

        vfixedFRAC_BITS
The number of bits in the fractional part of a fixed-point number. This constant has integral type.
