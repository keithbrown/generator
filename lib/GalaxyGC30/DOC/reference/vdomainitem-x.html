vdomainitem prefix



Types [3] Functions [21] Messages [1] Constants [1] Macros [3]

Types

vdomainitem

        typedef ... vdomainitem;
Contains the information needed to represent a single domain item and its contents. Domain items are referenced by a pointer to this type.

vdomainitemClass

        typedef ... vdomainitemClass;
Describes the behavior of a class of domain items. Domain item classes are referenced by a pointer to this type.

vdomainitemDomain

        typedef ... vdomainitemDomain;
Contains the information needed to represent a domain data object and its contents within a vdomainitem. This class is used exclusively by vdomainitem objects. Subclass this class if you must provide a specialized domain for a particular domain item to display.

Functions

vdomainitemClone

        vdomainitem * vdomainitemClone(
                vdomainitem * domainitem);
Creates a copy of the domain item domainItem and returns a pointer to the copy. This function is essentially a combination of vdomainitemCreate and vdomainitemCopy.

vdomainitemCopy

        void vdomainitemCopy(
                vdomainitem * domainitem,
                vdomainitem * targetItem);
Copies the information in domainItem into target so that the two domain items represent equivalent domain items but separate storage. Any objects created in domainItem are duplicated in target as well.

vdomainitemCopyInit

        void vdomainitemCopyInit(
                vdomainitem * domainitem,
                vdomainitem * targetItem);
Copies the information in domainItem into targetItem, so that the memory addressed by targetItem is a domain item equivalent to domainItem. targetItem should be a pointer to uninitialized memory; the Domain class initializes it as part of the copy.

vdomainitemCreate

        vdomainitem * vdomainitemCreate();
Creates a new domain item of the default domain item class and returns a pointer to the new domain item. (Memory is allocated by the Domain class.)

vdomainitemCreateOfClass

        vdomainitem * vdomainitemCreateOfClass(
                vdomainitemClass * clas);
Creates a new domain item of class clas and returns a pointer to the new domain item. (Memory is allocated by the Domain class.) The new domain item is created with a set of default attributes, determined by the class.

vdomainitemDestroy

        void vdomainitemDestroy(
                vdomainitem * domainitem);
Destroys the domain item represented by domainItem and all of its sub-objects that were allocated by the Domain class. This function attempts to free the storage directly occupied by the domain item only if it was allocated by vdomainitemCreate or vdomainitemCreateOfClass.

vdomainitemGetClass

        vdomainitemClass * vdomainitemGetClass(
                vdomainitem * domainitem);
Returns the Domain item class of the domain item domainItem.

vdomainitemGetData

        void * vdomainitemGetData(
                vdomainitem * domainitem);
Returns the generic data pointer attached to the domain item domainItem, which can be NULL.

vdomainitemGetDefaultClass

        vdomainitemClass * vdomainitemGetDefaultClass();
Returns the class used to create domain items by the vdomainitemCreate and vdomainitemInit functions. This class can be modified to alter the behavior of all domain items subsequently created by these functions.

vdomainitemGetDefaultDomainClass

        vdomainClass * vdomainitemGetDefaultDomainClass();
Returns the Domain class used by domain items when creating the domain for the item by vdomainCreateOfClass. This class can be modified to alter the behavior of all domains subsequently created by domain items.

vdomainitemGetDomain

        vdomain * vdomainitemGetDomain(
                vdomainitem * domainitem);
Returns the domain that the domain item domainItem is displaying.

vdomainitemGetItem

        void vdomainitemGetItem();
Returns the vdialogItem the domain item domainitem is built on.

vdomainitemGetLoadable

        void vdomainitemGetLoadable();
Returns the vloadable the domain item domainitem is built on.

vdomainitemGetView

        vdomainview * vdomainitemGetView(
                vdomainitem * domainitem);
Returns the domain view that the domain item domainItem belongs to.

vdomainitemInit

        void vdomainitemInit(
                vdomainitem * domainitem);
Makes the domain item addressed by domainItem into a valid domain item of the default domain data class in the same manner as vdomainitemInitOfClass.

vdomainitemInitOfClass

        void vdomainitemInitOfClass(
                vdomainitem * domainitem,
                vdomainitemClass * clas);
Makes the domain item addressed by domainItem into a valid domain item of class clas. The same address can be used in other calls to the Domain class as if it were created with vdomainitemCreate. A call to vdomainitemDestroy frees any internal objects allocated by the Domain class but does not attempt to free the structure itself.

vdomainitemLoad

        vdomainitem * vdomainitemLoad(
                vresource resource);
Reads a domain item from the resource resource and returns a pointer to its contents, which are allocated by the Domain class. An exception is generated if the resource has not been constructed properly.

vdomainitemLoadInit

        void vdomainitemLoadInit(
                vdomainitem * domainitem,
                vresource resource);
Reads a domain item from the resource resource and stores it in the structure addressed by domainItem, which should be uninitialized. An exception is generated if the resource has not been constructed properly.

vdomainitemSetData

        void vdomainitemSetData(
                vdomainitem * domainitem,
                void * data);
Sets the domain item domainItem's generic data pointer to data. This can be a pointer to whatever information the developer wants to attach to the domain item; it is not used by the Domain class. data can be NULL.

vdomainitemStartup

        void vdomainitemStartup();
Sets up the domain item portion of the Domain class for use. It has no effect if the startup has already been done.

vdomainitemStore

        void vdomainitemStore(
                vdomainitem * domainitem,
                vresource resource);
Writes the contents of the domain item domainItem into the resource resource. The domain item can be recovered from the resource at a later time (perhaps by another program) with vdomainitemLoad or vdomainitemLoadInit. The resource should have type vresourceDICT.

Messages

vdomainitemCREATE_DOMAIN

        <vdomainitemCREATE_DOMAIN>
Creates the domain for the domain item domainItem. The method is called when the domain item is initialized, and needs to create a domain. There is no associated function entry point for this message.

The default method for the default domain item class calls vdomainCreateOfClass with the default domain item domain class. Subclasses that override this method can perform any subclass-specific operations before or after the call to vclassSendSuper, depending on the desired behavior.

Macros

vdomainitemCLASS

        #define vdomainitemCLASS(...) ...
Contains the method and variable declarations for domain item classes. This macro expands to the contents of vdomainitemClass. SUPER is the type identifier of the class being declared (for example, vdomainitemClass). This type is used by the class's subclasses to refer to it. ITEM is the type identifier of instances of the class being declared (for example, vdomainitem). NOTIFY is the type identifier of the instance notification function (for example, vdialogItemNoteProc). DOMAIN is the type identifier for the domain viewed by the domain item (for example, vdomain). This macro should be used by domain item subclasses to declare the contents of the domain item class. This macro should come first in the subclass structure declaration.
