Types

vtransform

        typedef ... vtransform;
Represents a transformation matrix in fixed-point form, plus a set of flags that summarize the complexity of the transformation. The fixed-point elements correspond to matrix elements with the same name (refer to vmatrix). The structure members of a matrix are visible to clients of the Matrix Library but their particular representation is not. flags summarizes the complexity of the transformation and is the bitwise OR (|) of zero or more of the following constants:

vtransformTRANSLATE_X
tx!=0
vtransformTRANSLATE_Y
ty!=0
vtransformTRANSLATE
tx!=0 || ty!=0
vtransformINVERT_X
a<0
vtransformINVERT_Y
d<0
vtransformINVERT
a<0 || d<0
vtransformSCALE_X
abs(a)!=1
vtransformSCALE_Y
abs(d)!=1
vtransformSCALE
abs(a)!=1 || abs(d)!=1
vtransformROTATE
b!=0 || c!=0

Functions

vtransformFromMatrix

        void vtransformFromMatrix(
                const vmatrix * mtx,
                vtransform * transform);
Converts the matrix mtx to fixed-point form and stores the result in the transformation transform. This function also sets the flags of the transformation transform according to the complexity of the transformation.

vtransformIdent

        const vtransform * vtransformIdent();
Returns a pointer to the identity transformation. This pointer is valid throughout the lifetime of the process. The contents of this transformation should not be modified.

vtransformInvert

        void vtransformInvert(
                const vtransform * transform,
                vtransform * rtransform);
Computes the inverse of the transformation transform. Returns the inverted transformation in rtransform.

vtransformRound

        void vtransformRound(
                const vtransform * transform,
                vtransform * rtransform);
Removes any scaling from the transformation transform. Returns the rounded transformation in rtransform.

vtransformToMatrix

        void vtransformToMatrix(
                const vtransform * transform,
                vmatrix * mtx);
Converts the transformation transform to floating-point form and stores the result in mtx.

        vtransformROTATE
These constants are stored in the flags member of a vtransform structure and summarize the complexity of the transformation contained in the other members of the vtransform structure. Zero of more of these constants are bitwise ORed (|) together to represent a transformation complexity summary. These constants have integral type (refer to vtransform for more information).
