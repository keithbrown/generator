Types

vhelpdefault

        typedef ... vhelpdefault;
This help display no more than an subclass of vhelphypertext which is initialized with an instance of vwebbrowseDefault to which it will delegate the display of help data referenced by URLs. The display uses vwebbrowseGotoURL to pass instances of vurl (stored in vhelpref instances in the help request) to the default system browser.

vhelpdefaultClass

        typedef ... vhelpdefaultClass;
Contains message and variable bindings for objects of type vhelpdefault . Classes declared with this type contain the same set of methods and variables as the default system html help display class, but the behavior of methods and the contents of variables may vary. This class type is derived from vhelphypertext.

Macros

vhelpdefaultCLASS

        #define vhelpdefaultCLASS(...) ...
This macro contains the method declarations for the helpdefault class, this macro expands to the contexts of vhelpdefaultClass. SUPER<\SAMP> is the type identifier of the class being declared. This type is used by the class subclasses to refer to it. HELPDIS is the type identifier of the instances of the class being declared. This macro should be used by helpdefault subclasses to declare the contents of the helpdefault class. This macro should come first in the subclass structure declaration.
