vimageedtr

        typedef ... vimageedtr;
Contains the information needed to represent a single image editor and its contents. Image editors are referenced by a pointer to this type. An image editor provides a common interface to import, export, and edit images in any Galaxy application.

vimageedtrItem

        typedef ... vimageedtrItem;
Contains the information needed to represent a single image editor item and its contents. Image editors items are referenced by a pointer to this type. An image editor item provides a common interface to import, export, and edit images in a container form in any Galaxy application.

vimageedtrSetItemSelection

        void vimageedtrSetItemSelection(
                vimageedtrItem * editor,
                vimage * image,
                vimage * mask,
                int locx,
                int locy);
Sets the selection for the image editor item editor. image is the image to set as the selection and mask specifies the transparency, which may be NULL. The image editor takes ownership of image and mask. locx and locy specify the location to place the selection in the image being edited. This function should not be used to clear the selection or drop the current selection onto the image being edited. Instead, use vimageedtrDropSelection to drop the current selection onto the image being edited, and vimageedtrClearSelection to clear the current selection.

        void vimageedtrSetSelection(
                vimageedtr * editor,
                vimage * image,
                vimage * mask,
                int locx,
                int locy);
Sets the selection for the image editor editor. image is the image to set as the selection and mask specifies the transparency, which may be NULL. The image editor takes ownership of image and mask. locx and locy specify the location to place the selection in the image being edited. This function should not be used to clear the selection or drop the current selection onto the image being edited. Instead, use vimageedtrDropSelection to drop the current selection onto the image being edited, and vimageedtrClearSelection to clear the current selection.

Sets the show location option for the image editor editor. If value is TRUE, the image editor provides coordinate information (location, direction, and size) during drawing or editing operations.

vimageedtrSetViewRect

        void vimageedtrSetViewRect(
                vimageedtr * editor,
                const vrect * rect);
Sets the viewing rectangle of the image being edited in the image editor editor. The viewing rectangle is the portion of the image that is shown in the image editor window and is determined by the dialog item rectangle and the magnification (or bit size) of the image. The view rectangle should be pinned between 0, 0 and image->GetWidth, image->GetHeight where image is the image being edited.
