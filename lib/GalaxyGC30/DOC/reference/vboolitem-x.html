vboolitem

        typedef ... vboolitem;
A vboolitem is a vtypeditem derivative that represents a boolean value. The primary means by which vboolitem carries out it mission is by the use of a vformatBoolean object.

By default, the item is associated with a locale-specific global vformatBoolean object but other, non-shared, instances may be set by vboolitemSetFormat.

If the item has an associated vfield, as returned from vboolitemGetField, and vboolitemIsObserving] returns vFALSE, the field's value is set to value.

If the item has no associated field, it uses its vformatBoolean object to set various characteristics of the item, such as its content string, foreground color, and text justification.

vboolitemToggle

        void vboolitemToggle(
                vboolitem * item);
Most vtypeditem derived items support the vtypeditemINCREMENT message, but that message is most appropriate for continuously varying types. For the boolean type, only the two discreet states of true and false,/samp> are available; thus, vtypeditemIncrement is overridden to call on this method. Incrementing a vboolitem is like toggling its current value.
