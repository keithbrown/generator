vcomboCompareProc

        typedef ... vcomboCompareProc;
(Deprecated) Denotes a compare function that takes in two strings pointed by string1 and string2 and compares them (lexicographically by default). It generates three possible outcomes:
  • 1 if string1 is greater than string2
  • 0 if they are equal
  • -1 if string1 is less than string2.
In general, this vcomboCompareProc should be used internally as a pointer to vcomboCOMPARE message for overrides.

The type vcomboCompareProc is deprecated. It is used only by the deprecated class vcombo Use the vcombobox class instead of vcombo. Instead of providng a vcomboCompareProc, override the vcomboboxCOMPARE_VALUES message handler.

vcomboHookProc

        typedef ... vcomboHookProc;
(Deprecated) Denotes a hook function that is called when a designated combo box item handles an entry from its text box. combo is the combo box item checking on the string, and text is the string (held in the text box) being checked for validity. The hook function returns TRUE if the string is valid; FALSE otherwise. This hook function may call vcomboAppendText or vcomboAppendTextScribed to add a new text to combo box item's list if the string is valid.

Combo box items use pointers of this type to represent hook functions that are called when a combo box item's text box is completely entered. These pointers are defaulted as NULL, which implies no additions to combo box item's list and therefore returns TRUE. The API's vcomboSetHook and vcomboGetHook access the combo box item's hook pointer.

The type vcomboHookProc is deprecated. It is used only by the deprecated class vcombo. Use the vcombobox class instead of vcombo. Hook procs are not applicable to the vcombobox class.

        typedef ... vcomboNoteProc;

vcomboCOMPARE

        <vcomboCOMPARE>
(Deprecated) Compares the two strings pointed to by string1 and string2. It generates three possible outcomes:
  • 1 if string1 is greater than string2
  • 0 if they are equal
  • -1 if string1 is less than string2.
By default, this comparison is based on the function vcharCompare which compares two strings lexicographically. The message vcomboCOMPARE is automatically called by the message vcomboSORT_LIST.

The class vcombo is deprecated and only exists for backward compatibility with previous releases of Galaxy. Use vcombobox instead.
