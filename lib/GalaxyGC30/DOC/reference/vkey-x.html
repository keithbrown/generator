vkey prefix



Types [3] Functions [13] Constants [8]

Types

vkey

        typedef ... vkey;
An identifier used to indicate one of the keys on a keyboard or other key-based device. A vkey identifier can be converted into the vchar character set using vkeyToChar. Note that it is possible that some vkey values can not be converted to vchar values. See also the vkeyToChar and vkeyFromChar functions for caveats.

vkeyModifiers

        typedef ... vkeyModifiers;
A set of flags indicating which input modifiers were pressed. The following are valid vkeyModifiers flags:
  • vkeyMODIFIER_KEY_ALT
  • vkeyMODIFIER_KEY_CTRL
  • vkeyMODIFIER_KEY_SHIFT
  • vkeyMODIFIER_KEY_COMMAND
  • vkeyMODIFIER_BUTTON_LEFT
  • vkeyMODIFIER_BUTTON_MIDDLE
  • vkeyMODIFIER_BUTTON_RIGHT

vkeyStroke

        typedef ... vkeyStroke;
An object containing both a vkey and a vkeyModifiers.

vkeyDumpStroke

        void vkeyDumpStroke(
                vkeyStroke stroke,
                FILE * fp);
Prints the contents of the vkeyStroke stroke on the file stream in user-readable form. This function is intended for debugging and should not be used to store the contents of the vkeyStroke in recoverable form.

vkeyInternModifiers

        void * vkeyInternModifiers(
                vkeyModifiers modifiers);
Returns a pointer that can be used to uniquely identify the given vkeyModifiers. This is often used with dictionaries. For example: 

vname *binding;
vdict *dict;
vkeyModifiers modifiers;

...
binding = vdictLoad(dict, vkeyInternModifiers(modifiers));

vkeyInternStroke

                vkeyStroke stroke);
Returns a pointer that can be used to uniquely identify the given vkeyStroke. This is often used with dictionaries. For example: 

vname *binding;

...
binding = vdictLoad(dict, vkeyInternStroke(stroke));

vkeyParseModifiers

        vkeyModifiers vkeyParseModifiers(
                const vchar * s);
Parses a set of vkeyModifiers in string form. vkeyparseModifiers takes a vchar string of colon-delimited modifier tokens. For example: 

vstr *str;
vkeyModifiers modifiers;

str = vstrCloneScribed(vcharScribeLiteral("shift:control"));
modifiers = vkeyParseModifiers(str);
/* modifiers should be vkeyMODIFIER_SHIFT | vkeyMODIFIER_CTRL */
vstrDestroy(str);

vkeyToChar

        vchar vkeyToChar(
                vkey key);
Returns the vchar associated with the given vkey. Some vkeys might not be representable as a vchar, in which case the return value is undefined.

vkeyUninternModifiers

        vkeyModifiers vkeyUninternModifiers(

Returns the vkeyModifiers corresponding to the given internedModifiers. internedModifiers must be a value returned from a previous call to vkeyinternModifiers.
