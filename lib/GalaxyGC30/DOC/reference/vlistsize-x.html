Types [2] Functions [41] Messages [17] Constants [3] Macros [1]

Types

vlistsize

        typedef ... vlistsize;
Denotes an instance of the class derived from vlistsizeClass. This is a subclass of the vobject type, and is used to maintain the row heights and column width for a list view and list item. This can be used to create further subclasses for custom needs.

vlistsizeClass

        typedef ... vlistsizeClass;
The class definition for the list size object, it is derived from vobjectClass. This class is used in conjunction to a list view or list item to hold the row heights and column width. It can be subclassed to provide functionality past that provided by the default list size object in the List Manager.

Functions

vlistsizeCalcCellRange

        void vlistsizeCalcCellRange(
                vlistsize * size,
                int startRow,
                int startColumn,
                int width,
                int height,
                int * endRow,
                int * endColumn);
Calculates the range of cells that would be visible in an area with a given cell at the upper left. The range is calculated by determining the rectangle for cell (startRow, startColumn). The heights of rows beyond startRow added until the sum equals or exceeds height, or until the number of rows in the list size size are exhausted. An identical procedure is followed for column widths. The ending row and column are returned in endRow and endColumn. This function is implemented by sending the message vlistsizeCALC_CELL_RANGE to size.

vlistsizeClone

        vlistsize * vlistsizeClone(
                vlistsize * size);
Returns a pointer to a newly allocated list size object, and makes it a copy of the list size object size. This function is essentially a combination of vlistsizeCreate and vlistsizeCopy.

vlistsizeCopy

        void vlistsizeCopy(
                vlistsize * size,
                vlistsize * target);
Copies the information from size into target, so that the two list size objects represent equivalent objects but separate storage. The pointer in the object's data pointer field is copied, but the memory it points to is not duplicated.

vlistsizeCopyInit

        void vlistsizeCopyInit(
                vlistsize * size,
                vlistsize * target);
Copies the information in size into target so the memory addressed by target is a list size equivalent to size. target should be a pointer to uninitialized memory---the List Manager will initialize it as part of the copy. This function copies all of the stored size. This function is implemented by sending the message vobjectCOPY_INIT to size.

vlistsizeCreate

Creates a new list size object of class clas and returns a pointer to the new object, the memory for the new object is allocated by the List Manager. The new list object is created with a set of default attributes determined by the class.

vlistsizeSetColumnWidth

        void vlistsizeSetColumnWidth(
                vlistsize * size,
                int column,
                int width);
Sets the width of column to be width. The width of all columns can be set to the same value by passing vlistviewALL_COLUMNS in the column parameter. This sets the absolute width. Extra space is not added to width as it is with the vlistviewSetColumnWidth function. If the client never sets the column width, it is undefined until the client defines a default column width. If the column width is changed directly in the size structure with this call, the change does not propagate to the listview. It is the caller's responsibility to ensure that the visible display matches what is stored in the size structure.

If column is beyond the end of the list of stored sizes (the widths of columns 0 through 10 are set, and now you are setting the width of column 15), the default width is used for the width of the intervening columns. If this width has not been set by vlistsizeSetDefaultWidth, a /Module exception is generated. This function is implemented by sending the message vlistsizeSET_COLUMN_WIDTH to size.

vlistsizeSetDefaultHeight

        void vlistsizeSetDefaultHeight(
                vlistsize * size,
                int height);
Sets the default height to be used when an undefined size value is needed. The default height is used when no explicit height has ever been given for rows in the size set, and when a new row is added beyond the end of a size set. In the first case, the default height is used for all row heights, in the second case, the default height is used for the intervening rows between the end of the size list and the new row height being set. This function is implemented by sending the message vlistsizeSET_DEFAULT_HEIGHT to size.
