Types

vpoint

        typedef ... vpoint;
Represents a single coordinate whose x and y components are of integral type. Components of vpoint have at least 16 bits of precision. Galaxy services that operate on a single point typically pass an x and y value instead of a point structure, but some services that operate on multiple points pass a pointer to an array of points.

vpointFixed

        typedef ... vpointFixed;
Represents a single coordinate whose x and y components are of fixed-point type. Galaxy services that operate on a single point typically pass an x and y value instead of a point structure, but some services that operate on multiple points pass a pointer to an array of points.

vpointFloat

        typedef ... vpointFloat;
Represents a single coordinate whose x and y components are of floating-point type. Galaxy services that operate on a single point typically pass an x and y value instead of a point structure, but some services that operate on multiple points pass a pointer to an array of points.

vpointLong
