vinstGetSystemResources

        vresource vinstGetSystemResources();
(Deprecated) Returns a shared read-only resource that contains the Galaxy system resources. The Galaxy system resources are contained in the file vgalaxy1.vr, which is located with vinstFindResource.

Note that if this file is not found, a warning is issued and an empty in-memory resource is returned so that higher-level functions can recover gracefully. This function is used by higher-level Galaxy modules to access their resources; it should not be needed by applications. Applications can access their resources with vapplicationGetResources. Galaxy modules which have access to the Application class should use vapplicationGetSystemResources instead.

This deprecated entry point is included for backward compatibility. The recommended call is vinstallGetSystemResources.
