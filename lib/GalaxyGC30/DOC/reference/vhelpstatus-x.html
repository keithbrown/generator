Types

vhelpstatus

        typedef ... vhelpstatus;
This subclass of vhelpdisplay does not display text itself but instead announces the status of a object to any observers (refer to vobserver) who will then display the text. While any object subclassed from vobserver can observe these occurrences, most applications will use a vstatusbar. By default, all statusbars observe these messages from the default status help object (refer to vhelpregistry). This observation can be enabled or disabled with vstatusbarSetObserveStatusHelp.

This display expects the help data to take the form of a vhelptext. Status help occurrences are subclasses of voccurrence containing the help data as a string (refer to vhelpstatusOccurrence).

vhelpstatusClass

        typedef ... vhelpstatusClass;
Contains message and variable bindings for objects of type vhelpstatus . Classes declared with this type contain the same set of methods and variables as the status help display class, but the behavior of methods and the contents of variables may vary. This class type is derived from vhelpdisplay.

vhelpstatusOccurrence

        typedef ... vhelpstatusOccurrence;
Encapsulates the status help message as a string. Refer to voccurrence.

vhelpstatusOccurrenceClass

        typedef ... vhelpstatusOccurrenceClass;
Contains message and variable bindings for objects of type vhelpstatusOccurrence . Classes declared with this type contain the same set of methods and variables as the status help occurrence display class, but the behavior of methods and the contents of variables may vary. This class type is derived from voccurrence.

vhelpstatusAnnounce

        void vhelpstatusAnnounce(
                vhelpstatus * helpdisplay,
                voccurrence * occurrence);
Broadcasts an occurrence object to all of the invoking object's observers. This method calls the vobservable method on each of its observers, passing along the occurrence. If the occurrence initially has a NULL announcer (returned by voccurrence), then the invoking object is temporarily registered as its announcer voccurrence.

vhelpstatusInstallObserver

        void vhelpstatusInstallObserver(
                vhelpstatus * helpdisplay,
                vobservable observer);
Adds the given object to the list of observers on the invoking object. The order of observers is not specified and may change from announcement to announcement. Also, if the same object is installed two or more times, then it will get that many notices of every occurrence.

vhelpstatusUninstallObserver

        void vhelpstatusUninstallObserver(
                vhelpstatus * helpdisplay,
                vobservable observer);
Removes the given object from the list of observers on the invoking object. The passed object must already have been installed as an observer, or Galaxy throws an exception. If the object was installed more than once, only one of its entries is removed.

Macros

vhelpstatusCLASS

        #define vhelpstatusCLASS(...) ...
This macro contains the method declarations for the helpstatus class. This macro expands to the contexts of vhelpstatusClass. SUPER<\SAMP> is the type identifier of the class being declared. This type is used by the class subclasses to refer to it. HELPDIS is the type identifier of the instances of the class being declared. This macro should be used by helpstatus subclasses to declare the contents of the helpstatus class. This macro should come first in the subclass structure declaration.

vhelpstatusOccurrenceCLASS

        #define vhelpstatusOccurrenceCLASS(...) ...
This macro contains the method declarations for the helpstatusOccurrence class. This macro expands to the contexts of vhelpstatusOccurrenceClass. SUPER<\SAMP> is the type identifier of the class being declared. This type is used by the class subclasses to refer to it. OCCURRENCE is the type identifier of the instances of the class being declared. This macro should be used by helpstatusOccurrence subclasses to declare the contents of the helpstatusOccurrence class. This macro should come first in the subclass structure declaration.
