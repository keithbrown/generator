Types

vmapping

        typedef ... vmapping;
Maintains an unordered collection of owned or unowned object instances, using interned names as the keys to the element values.

The elements of the mapping object must be object whose class derives from vinstance. The keys must be global vname values.

Whether an element object is owned or not by the mapping object is determined by how it is added to the mapping object. The Owned entry points imply that the caller will not destroy the object and that the mapping object can and will destroy the object when it itself is destroyed. The Shared entry points imply that the caller will eventually destroy the object itself, but not before the mapping object is destroyed (or else the behavior of the mapping object is undefined).

vmappingAnyKey

        typedef ... vmappingAnyKey;
Maintains an unordered collection of owned or unowned object instances, using arbitrary pointer values as the keys to the element values.

The elements of the mapping object must be object whose class derives from vinstance. The keys may be any pointer value.

Whether an element object is owned or not by the mapping object is determined by how it is added to the mapping object. The Owned entry points imply that the caller will not destroy the object and that the mapping object can and will destroy the object when it itself is destroyed. The Shared entry points imply that the caller will eventually destroy the object itself, but not before the mapping object is destroyed (or else the behavior of the mapping object is undefined).

vmappingAnyKeyIterator

        typedef ... vmappingAnyKeyIterator;
Provides a way of iterating over the elements in an any-key mapping object.

The iterator is started with a valid any-key mapping object. During any active loop of the iteration, either the key or the element element itself may be retrieved from the iterator.

vmappingClass

        typedef ... vmappingClass;
Describes the behavior of a class of name-key mapping objects.

vmappingHashFunc

        typedef ... vmappingHashFunc;
Denotes a hash function that returns an identifying value for a given pointer so that mapping keys can be categorized.

A function of this type should return a value for each pointer that is passed to it. This value must be the same for each identical pointer. Values should be as distinct as possible to obtain a good distribution.

vmappingIterator

        typedef ... vmappingIterator;
Provides a way of iterating over the elements in an name-key mapping object.

The iterator is started with a valid name-key mapping object. During any active loop of the iteration, either the key or the element element itself may be retrieved from the iterator.

vmappingLoadable

        typedef ... vmappingLoadable;
Maintains a mapping object which can be stored in and loaded from a resource.

All elements of these loadable mapping objects must be of a class derived from the vloadable class.

Messages

vmappingADD_ELEMENT

        <vmappingADD_ELEMENT>
Implements the adding of new elements to the mapping object, without regard to ownership.

Both Owned and Shared entry points use this method to add new elements. They then mark ownership or lack of ownership by calling vmappingAnyKey.

vmappingELEMENT_CLASS

        <vmappingELEMENT_CLASS>
The base class of all elements for the class of mapping objects.

vmappingHASH_FUNC

        <vmappingHASH_FUNC>
The hash function used to categorize keys for the class of mapping objects.

vmappingREMOVE_ELEMENT

        <vmappingREMOVE_ELEMENT>
Implements the removing of the element with the specified key from the mapping object.

If the element was owned by the sequence object, then it and its ownership is returned to the caller. Otherwise, NULL is returned.
