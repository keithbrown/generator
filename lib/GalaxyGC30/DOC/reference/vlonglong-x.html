Types

vlonglong

        typedef ... vlonglong;
Denotes a double-width integral value. This datatype can be used wherever greater integral precision is needed than is provided by the base integral datatype on a given platform. Longlong numbers can be compared for range and equality, as well as decomposed into their integral parts.

vlonglongINCLUDED

        vlonglongINCLUDED
A preprocessor directive used to prevent multiple inclusion of the header file that defines the vlonglong datatype.

vlonglongSCALAR

        vlonglongSCALAR
If this preprocessor directive is TRUE the vlonglong type can be represented by a native scalar type known to the compiler. If set to FALSE a the vlonglong type is represented by a constructed type.
