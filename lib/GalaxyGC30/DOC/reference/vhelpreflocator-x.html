vhelpreflocator prefix



Types [2] Functions [9] Constants [2]

Types

vhelpreflocator

        typedef ... vhelpreflocator;
Help reference locators locate, generate, or use other objects to generate references to help. Help references take the form of instances of vhelpref.

The default implementation of vhelpreflocator does little more than build a vhelpref from the results of a call to vobject using the object of the help request.

vhelpreflocatorClass

        typedef ... vhelpreflocatorClass;
Contains method and variable bindings for objects of type vhelpreflocator . Classes declared with this type contain the same set of methods and variables as the default help reference class, but the behavior of methods and the contents of variables may vary. This class type is derived from vhelplocatorClass.

Functions

vhelpreflocatorCreate

        vhelpreflocator * vhelpreflocatorCreate();
Creates an instance of the default class of vhelpreflocator and returns a pointer to its contents. Space for the object is allocated dynamically. The Class Manager reclaims the storage occupied by the object when it is destroyed with vhelpreflocatorDestroy. Higher-level interfaces based on the core vhelpreflocator type can define a macro to this function for their Create function. This function essentially allocates space for a new instance and calls vhelpreflocatorInit on it.

vhelpreflocatorCreateOfClass

        vhelpreflocator * vhelpreflocatorCreateOfClass(
                vhelpreflocatorClass * helpreflocatorClass);
Creates an instance of class helpreflocatorClass and returns a pointer to its contents. Space for the object is allocated dynamically based on the object size of the class. The Class Manager reclaims the storage occupied by the object when it is destroyed with vhelpreflocatorDestroy. Higher-level interfaces based on the core vhelpreflocator type can define a macro to this function for their Create function. This function essentially allocates space for a new instance and calls vhelpreflocatorInitOfClass on it.

vhelpreflocatorDestroy

        vhelpreflocator * vhelpreflocatorDestroy(
                vhelpreflocator * helpreflocator);
Destroys the object helpreflocator, which must have been created or initialized by a Class Manager function. Interfaces based on the core object class can define a macro to this function for their Destroy function. This function is implemented by sending the message vinstanceDESTROY to helpreflocator.

vhelpreflocatorGetClass

        vhelpreflocatorClass * vhelpreflocatorGetClass(
                vhelplocator * helplocator);
Returns the class of the object helpreflocator. This pointer can be cast to higher-level classes if it is known that it is of the appropriate type. Class types can be verified with vclassIsKindOf.

vhelpreflocatorGetDefaultClass

        vhelpreflocatorClass * vhelpreflocatorGetDefaultClass();
This class can be modified to alter the behavior of subclasses of vhelpreflocator. This function can be called to explicitly register the default help reference locator class with the Class Manager.

vhelpreflocatorGetLocator

        vhelplocator * vhelpreflocatorGetLocator(
                vhelpreflocator * helpreflocator);
Returns the vhelplocator on which the help reference locator helpreflocator is built.

vhelpreflocatorInit

        void vhelpreflocatorInit(
                vhelpreflocator * helpreflocator);
Initializes the memory addressed by helpreflocator in such a way that the resulting structure can be used to store any objects whose classes inherit from vhelpreflocatorClass.

vhelpreflocatorInitOfClass

        void vhelpreflocatorInitOfClass(
                vhelpreflocator * helpreflocator,
                vhelpreflocatorClass * clas);
Initializes an object of class clas in the memory addressed by helpreflocator. Interfaces based on the core reference locator class can define a macro to this function for their Init function. This function simply stores the class in the class field of the object structure and sends it the message vinstanceINIT.

vhelpreflocatorLocateHelp

        vbool vhelpreflocatorLocateHelp(
                vhelpreflocator * helplocator,
                vhelprequest * request);
The vhelpref created is associated with the help request by vhelprequestSetHelpDataOwned.

Constants

        vhelpreflocatorCLASS
Use this macro to build subclasses of vhelpreflocator. Refer to vclass for more information on subclassing.

vhelpreflocatorINCLUDED

        vhelpreflocatorINCLUDED

#endif
