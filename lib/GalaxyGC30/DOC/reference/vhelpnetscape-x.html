Types

vhelpnetscape

        typedef ... vhelpnetscape;
This help display is no more than a subclass of vhelphypertext which is initialized with an instance of vwebbrowseNetscape to which it will delegate the display of help data referenced by URLs. The display uses vwebbrowseGotoURL to pass instances of vurl (stored in vhelpref instances in the help request) to the browser.

vhelpnetscapeClass

        typedef ... vhelpnetscapeClass;
Contains message and variable bindings for objects of type vhelpnetscape . Classes declared with this type contain the same set of methods and variables as the default Netscape help display class, but the behavior of methods and the contents of variables may vary. This class type is derived from vhelphypertext.

Macros

vhelpnetscapeCLASS

        #define vhelpnetscapeCLASS(...) ...
This macro contains the method declarations for the helpnetscape class, this macro expands to the contexts of vhelpnetscapeClass. SUPER<\SAMP> is the type identifier of the class being declared. This type is used by the class subclasses to refer to it. HELPDIS is the type identifier of the instances of the class being declared. This macro should be used by helpnetscape subclasses to declare the contents of the helpnetscape class. This macro should come first in the subclass structure declaration.
