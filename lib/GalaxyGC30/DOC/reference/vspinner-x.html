Types [4] Functions [66] Messages [16] Constants [1] Macros [1]

Types

vspinner

        typedef ... vspinner;
A subclass of vcontrol that allows the user to select a numeric value from a numeric range using increment/decrement buttons or an associated text item. Refer to the section vspinnerText.

vspinnerClass

        typedef ... vspinnerClass;
Describes the behavior of a class of vspinners. vspinner classes are referenced by a pointer to this type. The default vspinner class is a subclass of the default vcontrol.

vspinnerCompareProc

        typedef ... vspinnerCompareProc;
Denotes a function that compares two strings passed to it. It generates one of three possible outcomes: 10-1 In general, this vspinnerCompareProc notation should be used internally to represent vspinnerCOMPARE message overrides, and is designed specifically for text vspinners.

vspinnerText

        typedef ... vspinnerText;
Represents text vspinner. Text vspinners have an associated string list. Strings from this list are displayed instead of numbers.

vspinnerCOMPARE

        <vspinnerCOMPARE>
Compares the two strings passed to it. It generates one of three possible outcomes: 1 if the first string is greater than the second, 0-1 if the first string is less than the second.

This message is designed specifically for vspinnerTexts and is automatically sent to the vspinner when the message vspinnerSORT_LIST is sent. By default, this message calls vcharCompareBase to compare the strings.

vspinnerCREATE_TEXTITEM

Subclasses of vspinner that require changes to the text part should override this message's handler..
