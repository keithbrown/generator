vkeyitem prefix



Types [3] Functions [21] Messages [1] Constants [1] Macros [1]

Types

vkeyitem

        typedef ... vkeyitem;
A dialog item that exists simply to record keystrokes. A key item displays the last entered or set keystroke, including all modifiers. Use key items to request a keystroke from the user.

vkeyitemClass

        typedef ... vkeyitemClass;
Contains the message definitions for an keyItem class. This class type is derived from vdialogItemClass.

vkeyitemNoteProc

        typedef ... vkeyitemNoteProc;
Denotes a notification function that is called whenever a key down event occured in the key item. keyitem is the key item handling the event and event is the key down event.

Functions

vkeyitemClone

        vkeyitem * vkeyitemClone(
                vkeyitem * keyitem);
Creates a copy of the key item keyItem and returns a pointer to its contents. This function is essentially a combination of vkeyitemCreate and vkeyitemCopy.

vkeyitemCopy

        void vkeyitemCopy(
                vkeyitem * keyitem,
                vkeyitem * target);
Copies the information in keyItem into target, so that the two key items represent equivalent objects. target must be created with vkeyitemCreate or an equivalent function.

vkeyitemCopyInit

        void vkeyitemCopyInit(
                vkeyitem * keyitem,
                vkeyitem * target);
Copies the information in keyItem into targetRuler, so that the memory addressed by targetRuler is a key item equivalent to keyItem. targetRuler should be a pointer to uninitialized memory; the key item initializes it as part of the copy.

vkeyitemCreate

        vkeyitem * vkeyitemCreate();
Creates a new key item of the default class and returns its pointer. The key item is created with a set of default attributes. If a different set of attributes is desired, they must be set explicitly.

vkeyitemCreateOfClass

        vkeyitem * vkeyitemCreateOfClass(
                vkeyitemClass * clas);
Creates a new key item of class clas and returns a pointer to its contents. Storage for the key item is allocated by the Key item. The Key item reclaims the storage occupied by the key item when it is destroyed with vkeyitemDestroy. The new key item is created with a set of default attributes, determined by the class. The default attributes of the default window class are documented in the ``Standard Items'' chapter of the Galaxy Application Environment Programmer's Guide. This function essentially allocates space for a new key item and calls vkeyitemInitOfClass on it.

vkeyitemDestroy

        void vkeyitemDestroy(
                vkeyitem * keyitem);
Destroys the key item represented by keyItem.

vkeyitemGetClass

        vkeyitemClass * vkeyitemGetClass(
                vkeyitem * keyitem);
Returns the key item class for the key item keyItem.

vkeyitemGetData

        const void * vkeyitemGetData(
                vkeyitem * keyitem);
Returns the generic data pointer attached to the key item keyItem, which can be NULL.

vkeyitemGetDefaultClass

        vkeyitemClass * vkeyitemGetDefaultClass();
Returns the class used to create key items by the vkeyitemCreate and vkeyitemInit functions. This class should not be modified by applications, because the default key item class is not allocated on a per-application basis. Changes to this class are visible to all the applications in a given process and all the applications sharing the same display server. Server programs may alter this class to obtain custom behavior. This function can be called to explicitly register the default key item class with the Class Manager.

vkeyitemGetItem

        vdialogItem * vkeyitemGetItem(
                vkeyitem * keyitem);
Returns the dialog item to which the key item keyItem belongs.

vkeyitemGetKeystroke

        vkeyStroke vkeyitemGetKeystroke(
                vkeyitem * keyitem);
Returns the current keystroke for the key item keyItem. The current keystroke is the last keystroke set by entering a keystroke into the item, or by call vkeyitemSetKeystroke. It can return vkeyNULL_STROKE if a keystroke has never been set.

vkeyitemGetNotify

        vkeyitemNoteProc vkeyitemGetNotify(
                vkeyitem * keyitem);
Returns the notification procedure attached to the key item keyitem , or NULL if there is no notification procedure.

vkeyitemInit

        void vkeyitemInit(
                vkeyitem * keyitem);
Makes the key item addressed by keyItem a valid key item of the default class in the same manner as vkeyitemInitOfClass.

vkeyitemInitOfClass

        void vkeyitemInitOfClass(
                vkeyitem * keyitem,
                vkeyitemClass * clas);
Makes the key item addressed by keyItem a valid key item of class clas in the same manner as vkeyitemCreateOfClass, except that storage is not allocated for the key item. If the key item is passed to vkeyitemDestroy, any internally allocated storage is freed, but the key item itself is not freed. The storage addressed by keyItem need not be dynamically allocated.

vkeyitemLoad

        vkeyitem * vkeyitemLoad(
                vresource resource);
Reads a key item from the resource resource and returns a pointer to its contents. The contents are allocated by the key item. An exception is generated if the resource has not been constructed properly.

vkeyitemLoadInit

        void vkeyitemLoadInit(
                vkeyitem * keyitem,
                vresource resource);
Reads a key item from the resource resource and stores it in the structure addressed by keyItem, which should be uninitialized. An exception is generated if the resource has not been constructed properly.

vkeyitemSetData

        void vkeyitemSetData(
                vkeyitem * keyitem,
                const void * data);
Sets a key item's generic data pointer to data. This can be a pointer to whatever information the client wants to attach to the key item; it is not used by the key item. data can be NULL.

vkeyitemSetKeystroke

        void vkeyitemSetKeystroke(
                vkeyitem * keyitem,
                vkeyStroke keystroke);
Sets the keystroke to keystroke for the key item keyItem. This keystroke is displayed, and can be retrieved with vkeyitemGetKeystroke.

vkeyitemSetNotify

        void vkeyitemSetNotify(
                vkeyitem * keyitem,
                vkeyitemNoteProc noteProc);
Changes the instance notification procedure for the key item keyitem to noteProc. This function is called by vdialogNotifyItem when keystroke is entered by the user. Note that this function is not called for every event that is handled by the key item; if additional notification is required, you should override the appropriate event handler . noteProc may be NULL, in which case no instance notification is given. This function is implemented by sending the message vdialogSET_ITEM_NOTIFY to keyitem.

vkeyitemStartup

        void vkeyitemStartup();
Initializes the key dialog item module for use. This function has no effect if the this module has already been started up.

vkeyitemStore

        void vkeyitemStore(
                vkeyitem * keyitem,
                vresource resource);
Writes the contents of the key item keyItem into the resource resource. The key item can be recovered from the resource at a later time (perhaps by another application) with vkeyitemLoad or vkeyitemLoadInit. The resource should have type vresourceDICT.

Messages

vkeyitemSET_KEYSTROKE

        <vkeyitemSET_KEYSTROKE>
Sets the keystroke to keystroke for the key item keyItem. This keystroke is displayed, and can be retrieved with vkeyitemGetKeystroke.

Constants

vkeyitemINCLUDED

        vkeyitemINCLUDED
Defined when the header file vkeyitem.h is included.

Macros

vkeyitemCLASS

        #define vkeyitemCLASS(...) ...
Contains the method and variable declarations for vkeyitem classes. This macro expands to the contents of vkeyitemClass. SUPER is the type identifier of the class being declared; for examples, vkeyitemClass. This type is used by the class's subclasses to refer to it. ITEM is the type identifier of instances of the class being declared; for example, vkeyitem.
