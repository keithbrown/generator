Types

vhelptooltip

        typedef ... vhelptooltip;
This subclass of vhelpdisplay displays help text in a floating rectangle near an interface object. This display expects help requests to contain help data in the form of vhelptext instances. This text is typically very short.

Mechanisms internal to vwindow create vhelprequests for tooltip-type help. These are issued one second after the pointer comes to rest. Tooltips are removed when the mouse moves again, when a key or button is pressed, or after seven seconds. Tooltips are automatically placed near the item. Currently, vhelptooltip provides proper placement behavior for the following types of items:

  • vwindow
  • vdialog
  • vmenu
  • vdialogItem
  • vpaletteItem

vhelptooltipClass

        typedef ... vhelptooltipClass;
Contains message and variable bindings for objects of type vhelptooltip . Classes declared with this type contain the same set of methods and variables as the tooltip help display class, but the behavior of methods and the contents of variables may vary. This class type is derived from vhelpdisplay.

Macros

vhelptooltipCLASS

        #define vhelptooltipCLASS(...) ...
This macro contains the method declarations for the helptooltip class, this macro expands to the contexts of vhelptooltipClass. SUPER<\SAMP> is the type identifier of the class being declared. This type is used by the class subclasses to refer to it. HELPDIS is the type identifier of the instances of the class being declared. This macro should be used by helptooltip subclasses to declare the contents of the helptooltip class. This macro should come first in the subclass structure declaration.
