Types

vulongitem

        typedef ... vulongitem;
A vulongitem is a vtypeditem derivative that represents an unsigned long integer value. The primary means by which vulongitem carries out it mission is by the use of a vformatULong object.

By default the item is associated with a locale-specific global vformatULong object, but other, non-shared, instances may be set by vulongitemSetFormat.

vulongitemClass

        typedef ... vulongitemClass;
Contains method and variable bindings for objects of type vulongitem. Classes declared with this type contain the same set of methods and variables as the default unsigned Longitem class, but the behavior of methods and the contents of variables may vary. This class type is derived from vtypeditemClass.

vulongitemGetIncrement

        long vulongitemGetIncrement(
                vulongitem * item);
ULongItem's parent class, vtypeditem, supports the notion of incrementable, type-specific input items. The vulongitemIncrement method is called in response to user-initiated actions (like keystrokes) and its lone parameter, of type vtypeditemDirection, specifies whether the item's value is being lowered or raised. The absolute value of the increment can be obtained from this method. The default increment amount for vulongitem is one and is loaded out of the system resource file at startup time.
