Types [6] Functions [86] Messages [12] Constants [5] Macros [2]

Types

vruler

        typedef ... vruler;
Contains the information needed to represent a single ruler and its contents. Rulers are referenced by a pointer to this type. A ruler is a dialog item that allows users to manipulate the display of data in an associated dataspace.

vrulerClass

        typedef ... vrulerClass;
Contains method and variables bindings for objects of type vruler. Classes declared with this type contain the same set of methods and variables as the default ruler class, but the behavior of methods and the contents of variables may vary. This class type is derived from vdialogItemClass.

vrulerItem

        typedef ... vrulerItem;
Contains the information needed to represent a single ruler item and its contents. Ruler items are referenced by a pointer to this type.

vrulerItemClass

        typedef ... vrulerItemClass;
Contains method and variables bindings for objects of type vrulerItem. Classes declared with this type contain the same set of methods and variables as the default ruler item class, but the behavior of methods and the contents of variables may vary. This class type is derived from vobjectClass.

vrulerIterator

        typedef ... vrulerIterator;
Is used to iterate over the ruler items in an ruler. An iterator is an abstract index variable which may refer to a particular ruler item. Each iterator refers to a particular ruler and only behaves correctly if used with that ruler. An iterator becomes invalid if any ruler items are added, removed or arranged in the icon view to which it is attached.

vrulerNoteProc

        typedef ... vrulerNoteProc;
Denotes a notification function that is called when a designated ruler handles an event of importance. ruler is the ruler handling the event, and event is the event being handled. If event is NULL, then the event is a ruler item event, where item is the item handling the event, and message is the item event type.

vrulerSetNotify

        void vrulerSetNotify(
                vruler * ruler,
                vrulerNoteProc noteProc);
Changes the notification procedure for the ruler ruler to noteProc. Whenever the ruler handles an event of importance, this procedure is called with the ruler, event ruler item, and message as arguments. The ruler item and message fields apply only for ruler item events, in which case event is NULL. The message field can be:
vrulerITEM_ADD
The ruler item was added.
vrulerITEM_DOUBLE
The ruler item was double-clicked.
vrulerITEM_HIDE
The ruler item was hidden.
vrulerITEM_MOVE_END
The ruler item was just moved.
vrulerITEM_MOVE_START
The ruler item is about to be moved.
vrulerITEM_REMOVE
The item was taken off ruler and will be destroyed.
vrulerITEM_RESIZE_END
The ruler item was just resized.
vrulerITEM_RESIZE_START
The ruler item is about to be resized.
vrulerITEM_SELECT
The ruler item was selected.
vrulerITEM_SHOW
The ruler item was shown.
vrulerITEM_WILL_REMOVE
The item is about to be taken off ruler and destroyed.
vrulerRESIZE
The ruler was resized.
vrulerSCROLL_END
The ruler was just scrolled.
vrulerSCROLL_START
The ruler is about to be scrolled.
noteProc can be NULL, in which case no notification is given.

Macros

vrulerCLASS

        #define vrulerCLASS(...) ...
Contains the method and variable declarations for ruler classes; this macro expands to the contents of vrulerClass.

SUPER is the type identifier of the class being declared; for example, vrulerClass. This type is used by the class's subclasses to refer to it. RULER is the type identifier of instances of the class being declared; for example, vruler. ITEM is the type identifier of ruler items in the ruler; for example, vrulerItem.

This macro should be used by ruler subclasses to declare the contents of the ruler class. This macro should come first in the subclass structure declaration.

vrulerItemCLASS

        #define vrulerItemCLASS(...) ...
Contains the method and variable declarations for ruler item classes; this macro expands to the contents of vrulerItemClass. SUPER is the type identifier of the class being declared (for example, vrulerItemClass). This type is used by the class's subclasses to refer to it. ITEM is the type identifier of instances of the class being declared (for example, vrulerItem). This macro should be used by ruler item subclasses to declare the contents of the ruler item class. This macro should come first in the subclass structure declaration.
