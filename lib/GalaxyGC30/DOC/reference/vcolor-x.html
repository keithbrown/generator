Types

vcolor

        typedef ... vcolor;
The basic unit of drawing color. It serves as an object that can be examined, passed, received, and even used to draw (by the Drawing class). All interned vcolor objects are stored in a shared global table and thus, may not be changed or modified in any way. All private colors can be manipulated by the application and must be destroyed by the application.

vcolorClass

        typedef ... vcolorClass;
(Deprecated) The color model that a color specification belongs to. There is currently one class for each existing color model. The available color classes are RGB, HLS, and CMY. Future releases will provide other color classes, such as CIE (human eye, color-perception classification) and HVC. This is deprecated in favor of vcolorModel.

vcolorConvertSpec

        void vcolorConvertSpec(
                vcolorSpec * spec,
                const vcolorModel * model);
Converts the specification spec into the color model model, preserving the visual appearance of the color as much as possible. The supported color models are RGB, HLS, and CMY, which can be retrieved by vcolorGetRGBModel, vcolorGetHLSModel, and vcolorGetCMYModel, respectively. The deltas are also converted with a fractional error. Manual delta setting (by vcolorSetSpecDeltaRGB or equivalent, depending on the color model) for new models is recommended for higher delta precision.

vcolorCreate

        vcolor * vcolorCreate();
Creates a new private color with the RGB model. The color is initialized and usable, although the component values are undefined. vcolorDestroy must be used on vcolors that are created with vcolorCreate. An associated color specification is also created on this call. The delta of the associated color specification defaults to value vcolorCOMPONENT_MAX divided by 4 of type vcolorDELTA_RGB_DISTANCE. The maximum dot dither area of this associated color specification is set to 4.

vcolorCreatePalette

        vcolorPalette * vcolorCreatePalette();
Creates a new color palette. The palette is initialized and usable, although the number of components and their values are undefined. vcolorDestroyPalette must be used on palettes that are created with vcolorCreatePalette. Set the size of the palette with vcolorSetPaletteSize. Use the standard vcolorSet routines, such as vcolorSetSpecRGB, to set the individual color specifications retrieved from the palette with vcolorGetPaletteSpecAt.

vcolorCreateSpec

        vcolorSpec * vcolorCreateSpec();
Creates a new color specification with the RGB model. The specification is initialized and usable, although the component values are undefined. vcolorDestroySpec must be used on vcolorSpecs that are created with vcolorCreateSpec. The delta of the created specification defaults to value vcolorCOMPONENT_MAX divided by 4 of type vcolorDELTA_RGB_DISTANCE. The maximum dot dither area also sets to 4.

vcolorCreateSpecSpecific

        vcolorSpec * vcolorCreateSpecSpecific(
                const vcolorModel * model);
Creates a new color specification with the color model model. The specification is initialized and usable, although the component values are undefined. vcolorDestroySpec must be used on vcolors that are created with vcolorCreateSpecSpecific. The supported color models are RGB, HLS, and CMY, which can be retrieved by vcolorGetRGBModel, vcolorGetHLSModel, and vcolorGetCMYModel, respectively.

The delta of the created specification defaults to value vcolorCOMPONENT_MAX divided by 4 in all components of type vcolorDELTA_RGB_DISTANCE, vcolorDELTA_HLS, or vcolorDELTA_CMY_DISTANCE, depending on the color model. The maximum dot dither area also sets to 4.

vcolorDestroy

        void vcolorDestroy(
                vcolor * color);
Releases the color color and any resources it may have been using. Use caution when calling this routine. The color that is released cannot be in use by the Drawing class or attached to any other object, such as a dialog item. Further, it is not necessary to call this routine if the color is active for the life of the program and it is not a private color (creation was made by vcolorInternSpec); Galaxy releases it on shutdown. Private colors, created by vcolorCreate or vcolorCreateSpecific, must be released by the application.

vcolorCOMPONENT_MAX

        vcolorCOMPONENT_MAX
An integral value representing the maximum possible value of any component of a color specification. The minimum is zero (0), and the maximum can be assumed to be greater than the minimum.

An integral value representing the minimum possible value of any component of a color specification. This minimum is zero (0), and the maximum can be assumed to be greater than the minimum.

vcolorDELTA_CMY

        vcolorDELTA_CMY
Tag used to indicate the delta type ``delta cube'' of CMY components in a given specification in CMY space. The default value for a CMY delta cube is 25 percent of vcolorCOMPONENT_MAX in each of CMY components.

vcolorDELTA_CMY_DISTANCE

        vcolorDELTA_CMY_DISTANCE
Tag used to indicate the delta type ``delta sphere'' of all CMY components in a given specification in CMY space. The default value for a CMY delta distance is 25 percent of vcolorCOMPONENT_MAX.

vcolorDELTA_HLS

        vcolorDELTA_HLS
Tag used to indicate the delta type ``delta sector'' of HLS components in a given specification in HLS space. The default value for a HLS delta sector is 25 percent of vcolorCOMPONENT_MAX in each of HLS components.

vcolorDELTA_HLS_DISTANCE

        vcolorDELTA_HLS_DISTANCE
Tag used to indicate the delta type ``delta distance'' of HLS components in a given specification in HLS space. The default value for a HLS delta distance is 25 percent of vcolorCOMPONENT_MAX.

vcolorDELTA_RGB

        vcolorDELTA_RGB
Tag used to indicate the delta type ``delta cube'' of RGB components in a given specification in RGB space. The default value for a RGB delta cube is 25 percent of vcolorCOMPONENT_MAX in each of RGB components.

vcolorDELTA_RGB_DISTANCE

        vcolorDELTA_RGB_DISTANCE
Tag used to indicate the delta type ``delta sphere'' of RGB components in a given specification in RGB space. The default value for a RGB delta distance is 25 percent of vcolorCOMPONENT_MAX.
