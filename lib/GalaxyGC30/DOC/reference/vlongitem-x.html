vlongitem

        typedef ... vlongitem;
A vlongitem is a vtypeditem derivative that represents a long int value. The primary means by which vlongitem carries out it mission is by the use of a vformatLong object.

By default, the item is associated with a locale-specific global vformatLong object, but other, non-shared, instances may be set by vlongitemSetFormat.

vlongitemClass

        typedef ... vlongitemClass;
Contains method and variable bindings for objects of type vlongitem. Classes declared with this type contain the same set of methods and variables as the default Longitem class, but the behavior of methods and the contents of variables may vary. This class type is derived from vtypeditemClass.

vlongitemGetIncrement

        long vlongitemGetIncrement(
                vlongitem * item);
LongItem's parent class, vtypeditem, supports the notion of incrementable type-specific input items. The vlongitemIncrement method is called in response to user-initiated actions (like keystrokes) and its lone parameter, of type vtypeditemDirection, specifies whether the item's value is being lowered or raised. The absolute value of the increment can be obtained from this method. The default increment amount for vlongitem is one (1) and is loaded out of the system resource file at startup time.
