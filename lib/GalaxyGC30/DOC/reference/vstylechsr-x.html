Types [3] Functions [15] Messages [2] Constants [1] Macros [1]

Types

vstylechsr

        typedef ... vstylechsr;
A style chooser provides a common interface to the user for choosing text styles used in text data objects. A style chooser can be created by any Galaxy application and is the recommended method for requesting a text style from the user.

vstylechsrApplyFunc

        typedef ... vstylechsrApplyFunc;
Denotes an apply function that is called when a style has been chosen from the style chooser. style specifies the style that was chosen. The apply function should return TRUE if the style was acceptable, in which case the style chooser closes if appropriate, or FALSE if the style is not acceptable, in which case the style chooser does not close under any circumstances.

vstylechsrClass

        typedef ... vstylechsrClass;
Describes the behavior of a class of style choosers. The style chooser class controls style application. Style chooser classes are referenced by a pointer to this type.

Functions

vstylechsrClose

        void vstylechsrClose(
                vstylechsr * chooser);
Closes the style chooser styleChooser so that it is no longer visible. This function has no effect on a style chooser that is already closed.

vstylechsrCreate

        vstylechsr * vstylechsrCreate();
Creates a new font chooser of the default style chooser class and returns a pointer to the new style chooser. (Memory is allocated by the Style Chooser.)

vstylechsrCreateOfClass

        vstylechsr * vstylechsrCreateOfClass(
                vstylechsrClass * clas);
Creates a new style chooser of class clas and returns a pointer to the new style chooser. (Memory is allocated by the Style Chooser.) The new style chooser is created with a set of default attributes determined by the class.

vstylechsrDestroy

        void vstylechsrDestroy(
                vstylechsr * styleChooser);
Destroys the style chooser represented by styleChooser and all of its sub-objects that were allocated by the Style Chooser. This function attempts to free the storage directly occupied by style chooser only if it was allocated by vstylechsrCreate or vstylechsrCreateOfClass .

vstylechsrGetApply

        vstylechsrApplyFunc vstylechsrGetApply(
                vstylechsr * styleChooser);
Returns the apply function for the style chooser styleChooser, or NULL if there is no apply function.

vstylechsrGetClass

        vstylechsrClass * vstylechsrGetClass(
                vstylechsr * styleChooser);
Returns the style chooser class of the style chooser styleChooser.

vstylechsrGetData

        const void * vstylechsrGetData(
                vstylechsr * styleChooser);
Returns the generic data pointer attached to the style chooser styleChooser, which can be NULL.

vstylechsrGetDefaultClass

        vstylechsrClass * vstylechsrGetDefaultClass();
Returns the class used to create a style chooser by the vstylechsrCreate function. This class can be modified to alter the behavior of all style choosers subsequently created by these functions.

vstylechsrGetDialog

        vdialog * vstylechsrGetDialog(
                vstylechsr * styleChooser);
Returns the dialog upon which the style chooser styleChooser is built. Dialog Manager operations on this dialog affect the style chooser it came from. All Dialog Manager operations have effect, but some may interfere with the style chooser's behavior.

vstylechsrGetStyle

        vtextStyle * vstylechsrGetStyle(
                vstylechsr * styleChooser);
Returns the current text style for the style chooser styleChooser. The current text style is the style that is constructed in the style chooser.

vstylechsrOpen

        void vstylechsrOpen(
                vstylechsr * chooser);
Opens the style chooser styleChooser so that it is visible at its current position and size on the screen. This function has no effect if styleChooser is already open.

vstylechsrSetApply

        void vstylechsrSetApply(
                vstylechsr * styleChooser,
                vstylechsrApplyFunc applyFunc);
Changes the apply function for the style chooser styleChooser to applyFunc. Whenever a style is chosen from the style chooser, the apply function is called, passing the style that was constructed. applyFunc can be NULL, in which case no apply notification is given.

vstylechsrSetData

        void vstylechsrSetData(
                vstylechsr * styleChooser,
                const void * data);
Sets a style chooser's generic data pointer to data. This can be a pointer to whatever information the client wants to attach to the style chooser---it is not used by the Style Chooser. data can be NULL.

vstylechsrSetStyle

        void vstylechsrSetStyle(
                vstylechsr * styleChooser,
                vtextStyle * style);
Sets the current style to style for the style chooser styleChooser. The current style is the style that is constructed in the style chooser.

vstylechsrStartup

        void vstylechsrStartup();
Initializes the Galaxy Style Chooser module. You must do this before creating any style choosers.

Messages

vstylechsrHOOK_APPLY_STYLE

        <vstylechsrHOOK_APPLY_STYLE>
Applies the style style, currently selected in the style chooser styleChooser. This method should return vTRUE if selected style is accepted and FALSE otherwise. The default style chooser class method simple calls style chooser notification function, set by SetHookApplyStyleFunc or returns TRUE if no notification function is selected.

vstylechsrSET_STYLE

        <vstylechsrSET_STYLE>
Sets the current style to style for the style chooser styleChooser. The current style is the style that is constructed in the style chooser.
