Types [4] Functions [39] Messages [5] Constants [15] Macros [1]

Types

vprocess

        typedef ... vprocess;
Contains the information necessary to represent a single process. Objects of this type have classes derived from vprocessClass. A process is an independently running application. The process contains the information necessary for when a process terminates, or to cause termination of that process. Processes can be only created dynamically.

vprocessArgumentTemplate

        typedef ... vprocessArgumentTemplate;
Data structure used to define command-line arguments. It contains a structure member argument that is a vstr* which indicates the argument.

vprocessClass

        typedef ... vprocessClass;
Contains method and class variable bindings for objects of type vprocess. An instance of the process class provides a platform-independent mechanism for launching processes from Galaxy.

vprocessEnvironmentTemplate

        typedef ... vprocessEnvironmentTemplate;
Data structure used to define environment variables for the process. The structure contains two members, variable and value. Variable is a string which is the environment's varaible name and value is also string containing the environment variable's value.

vprocessCancelWaitForTermination

        void vprocessCancelWaitForTermination(
                vprocess * process);
Cancels the wait for termination of a process. If a process has been terminated and the application is waiting for the termination of that process, this call will indicate that the wait for termination should be cancelled. In other words, this method sets a flag on the process saying that the application is not waiting for the termination of the process and may continue on; vprocess will then let the process terminate without notification.

vprocessGetEnvironmentVariables

        const vprocessEnvironmentTemplate * vprocessGetEnvironmentVariables(
                vprocess * process);
Returns the environment variables set when the process is started. An array of vprocessEnvironmentTemplate structures is returned. The end of the array is signified by a NULL array element. The structure contains two members, variable and value.

Variable is a string which is the environment's varaible name and value is also string containing the environment variable's value. The array is owned by the Process class and should not be destroyed unless the environment variables were set with the owned entry point, and then only when the process instance is being destroyed.

        void vprocessSetArguments(
                vprocess * process,

Sets the arguments passed as command-line arguments when the process is started. An array of vprocessArgumentTemplate structures is passed as an argument.

Sets the arguments passed as command-line arguments when the process is started. An array of vprocessArgumentTemplate structures is passed as an argument.

The end of the array is signified by a structure with a NULL argument entry. Ownership of the array is taken by the process. The structure member argument is a vstr* which indicates the argument. The arguments are passed in the same order they occupy in the array.

vprocessSetCreationFlags

        void vprocessSetCreationFlags(
                vprocess * process,
                int flags);
Sets the flags which specify how the process is to be created. The possible values for flags are vprocessCREATION_DETACHED and vprocessCREATION_CONSOLE. Detached indicates a program which has a user-interface and does not do i/o through a console window. Console indicates a program which does do i/o through a console window.

vprocessSetEnvironmentVariables

        void vprocessSetEnvironmentVariables(
                vprocess * process,
                vprocessEnvironmentTemplate * args);
Sets the environment variables to be used by the process when started. An array of vprocessEnvironmentTemplate structures is passed as an argument. The end of the array is signified by a NULL array element. The structure contains two members, variable and value. Variable is a string which is the environment's varaible name and value is also string containing the environment variable's value. Ownership of the array and its strings is that of the caller. The caller should only destroy the array and its contained strings when the process is being destroyed.

The end of the array is signified by a NULL array element. The structure contains two members, variable and value. Variable is a string which is the environment's varaible name and value is also string containing the environment variable's value. The array is owned by the Process class and should not be destroyed.

vprocessStart

        vbool vprocessStart(
                vprocess * process);
Begins execution of the process described by the program name and path attributes. The program name, path, working path, priority, and type must be set before a call to start. The program path, working path, priority, and type will all be set to default values of current path, program path, and normal if not set before calling start.

vprocessStartup

        void vprocessStartup();
Starts up the process class. Must be called before using the process class.

vprocessTerminate

        vbool vprocessTerminate(
                vprocess * process,
                int status);
Terminates execution of the program running within the process. The state of the process will be set to vprocessSTATE_INACTIVE if the operation succeeds. Likewise a value of TRUE will be returned on success. The status argument is ignored on all platforms besides Win32, on which case it is directly passed to the Win32 call TerminateProcess.

vprocessWaitForTermination

        int vprocessWaitForTermination(
                vprocess * process,
                vtimestamp * timeout,
                int * status);
Requests termination of a process and waits for its actual termination. The wait for termination will stop after the amount of time specified in the vtimestamp timeout. The value of status is platform dependent and is the appropriate process exit status for the current platform. The value returned will be one of:

  • vprocessWAIT_TIMED_OUT
  • vprocessWAIT_CANCELLED
  • vprocessWAIT_FAILED
  • vprocessWAIT_SUCCESS

vprocessPRIORITY_IDLE

vprocessSTATE_ALL

        vprocessSTATE_ALL
Mask for process state flags.

vprocessSTATE_INACTIVE

Indicates that the process is not active, meaning it has either not been started or has terminated. This value is returned by vprocessGetState

vprocessWAIT_CANCELLED

        vprocessWAIT_CANCELLED
Constant used to indicate that the state of waiting for process termination has been cancelled. This constant is returned by vprocessWaitForTermination. The routine which causes this action is vprocessCancelWaitForTermination.

vprocessWAIT_FAILED

        vprocessWAIT_FAILED
Constant used to indicate that the state of waiting for process termination has failed. This constant is returned by vprocessWaitForTermination.

vprocessWAIT_SUCCESS

        vprocessWAIT_SUCCESS
Constant used to indicate waiting for process termination has succeeded. This constant is returned by vprocessWaitForTermination.

vprocessWAIT_TIMED_OUT

        vprocessWAIT_TIMED_OUT
Constant used to indicate that the state waiting for process termination timed out. This constant is returned by vprocessWaitForTermination.

Macros

vprocessCLASS

        #define vprocessCLASS(...) ...
Contains method and variable bindings for objects of type vprocess. Classes declared with this type contain the same set of methods and variables as the default process class, but the behavior of methods and the contents of variables may vary. This class type is derrived from vinstance.
