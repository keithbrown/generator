Accessing Help from VRB

 Help for the Visual Resource Builder is available in the following ways:
  • Context-sensitive Help
    Pressing F1 provides context-sensitive help for the currently focused component in the Visual Resource Builder. Not all components in are focusable. Use Shift+F1 to access help on components that are not focusable.

  • What's This?
    Pressing Shift+F1 provides help on individual components of the Visual Resource Builder. After pressing Shift+F1, the cursor changes to the Help Cursor. Click on a component with the Help Cursor to access help on that component.

  • Help Menu
    Use the Help Menu in the Visual Resource Builder for quick access to the following:

    • "Contents
      Accesses the Help Map. Use the Help Map to navigate to any area of online help.

    • Getting Started
      Accesses the Getting Started page. Use the links to get up and going quickly with Galaxy.

    • Programmer's Guide
      Accesses the Programmer's Guide.

    • Reference
      Accesses the Reference Manual.

    • Tutorial
      Accesses the Tutorials.

    • Code Samples
      Accesses the Code Samples.

    • Help On Help
      Accesses this help file.

    • Search...
      Accesses the search engine. Use the search engine to search for specific topics of help. [The search engine is currently not implemented.]

    • What's This
      Provides help on individual items in the Visual Resource Builder. Invoking What's This changes the cursor to the Help Cursor. Click on an item with the Help Cursor to access help on that item.

    • About the Visual Resource Builder...
      Provides information on the Visual Resource Builder.

  • Help Browser Preferences
    You can specify the System Default browser, Netscape Navigator, or Internet Explorer to display online help. To specify a help browser, select Edit:Preferences...; then in the Preferences Editor, select Project:Help. Note that the browser of choice must be installed on your system. You can also specify whether the Visual Resource Builder will automatically raise the browser when accessing help.
