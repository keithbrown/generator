A toolbar is a container subclass that displays icons representing an object or action available to a user.

Open the Toolbar Editor by clicking on a toolbar in a resource, or by double-clicking on a toolbar added to a dialog in the Dialog Editor.

Use the Toolbar Editor to:

  • Add buttons to a toolbar
  • Add an image to a toolbar button
  • Remove buttons from an toolbar
  • Arrange the ordering of buttons on a toolbar

Adding Buttons to a Toolbar

To add a new button to a toolbar, select the Toolbar:New Button menu option, or click the New Button button. A new button with the tag "ToolbarButtonItem" and an empty title is added to the toolbar.

Adding an Image to a Toolbar Button

To add an image to a toolbar button, select the button and choose an icon from the menu on the Default Icons button.

You can edit an image or create a new image using the Image Editor. Locate the Image item in the Properties List. Make sure its checkbox is checked and click the ellipsis (...) button to open the Image Editor.

Removing Buttons from a Toolbar

To remove a button from a toolbar, select the button you want to remove. Then, click the Remove Item button. Alternately, you can use Clear or Cut from the Edit menu or the Edit toolbar.

Arranging Buttons on a Toolbar

You can control the order in which the choices are displayed on a toolbar. Select the button you want to move, and then drag it to its new position on the toolbar.
