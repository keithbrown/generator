Connections Tutorial

Dialog Editor
Use the Spring Editor, in conjunction with the Dialog Editor to precisely edit spring and strut connections. Connections specify the relative positioning of items in a dialog.

For further information on geometry management using connections, refer to the Connections Tutorial.

Use the Spring Editor to:

  • Select connections for editing
  • Access information on connections
  • Change the type of the selected connection
  • Specify a value for selected connections

Spring Editor Map

Connections Window
Shows the connections for selected items in the Dialog Editor. The Connections Window contains spring and struts that correspond to the inner and outer connections of selected items in the Dialog Editor.

Spring Editor Buttons
Sets the type (natural size, spring, or strut) for selected connections. Refer to the Connections Tutorial for information on types of connections.

Connection Value Spinner
Sets the length of selected connections.

Positioning Information
Provides information about selected connections.

Selecting Connections for Editing

The Spring Editor allows you to select one or more connections for editing. To select connections for editing:
  1. Select one or more dialog items in the Dialog Editor.

  2. In the Connections Window, select an inner or outer connection. This selects the corresponding connection(s) for the selected dialog item(s).
For example, to specify that a series of dialog items are all 10 pixels from the left edge of the dialog, select the items in the Dialog Editor. Then select the left outer connection in the Connections Window and specify a value of 10 in the Connection Value Spinner.

You can also select connections using right mouse-click in the area next to Positioning Information in the Spring Editor. Right mouse-click opens pop-up menus that allow you to specify the item to select, which connection to select, and the source and destination of the connection.

Accessing Information on Connections

The Spring Editor provides the following information on selected connections.
  • Tag: Tag name of the item for the selected connection

  • Which: Which connection is selected

  • To: Destination of the connection

  • To Edge: Location of the connection on its destination

Changing the Type of Connections

To change the type of selected connections:

  1. Select the items in the Dialog Editor that you want to change the connections for.

  2. In the Connections Window, do any of the following:

    • Double-click on the connection corresponding to the ones that you want to change. When you double-click on a connection, a spring changes to a strut; a strut changes to a spring.

    • Select the Natural Size Toggle to toggle the selected connection(s) between natural length and fixed length.

    • Select the Spring Toggle to toggle the selected connection(s) between struts and fixed-base plus springs.

    • Select the Strut Toggle to toggle the selected connection(s) between fixed-length struts and locked fixed-length struts. A locked fixed-length strut is an editing aid that prevents an item from moving in the editor. The item is not "locked" in the actual application.

    • Use right mouse-click to open a pop-up menu for specifying options for the selected connection(s).

Specifying Values for Connections

The value for a connections specifies the length of the connection, in pixels. To set the value for connections:

  1. Select the items in the Dialog Editor that you want to set the value for.

  2. Specify the value in the Value Spinner. Either scroll to a value or enter the value directly in the spinner's value field.
