Notebooks
Use the Notebook Editor to create notebooks for your application. Notebooks are dialog items that organize and present information to a user so the user can find, display, and edit the information quickly and easily. Notebooks are often used in dialogs that implement preferences or other user settings. A notebook resembles a physical notebook in appearance and performance.

Open the Notebook Editor by double-clicking on a notebook in the Dialog Editor.

Refer to the Notebook online documentation for more details on manipulating notebooks.

Glossary

The following terms are used in conjunction with Notebooks:

Folio
Status line for a notebook that displays the total number of pages in a notebook and the page number of the active page.

Major Tabs
Tabs drawn for top-level pages and sections in a notebook.

Minor Tabs
Tabs drawn for second level pages or sections in a notebook. If a section is at the top level of a notebook tree, and is the active selection (is visible), then tabs are drawn for each page and section contained within the section.

Notebook
Dialog item that presents information to a user in a series of pages and sections. Pages and sections can optionally display tabs which allow quick access to the page or section. Notebook items are modeled after physical notebooks. Notebooks contains all the pages and sections.

Page
Unit of information displayed by a notebook and is comparable to a sheet from a physical notebook. A page is a subclass of container and can hold any number of dialog items. Pages are typically used to present information to a user that the user can view or edit. A page represents the information presented on one sheet from a physical notebook. Each page contains a tab which provides the user access to the page.

Page Buttons
Buttons displayed with a notebook that allow the user to step forward or backward through the pages of a notebook.

Section
Subclass of a page that holds pages or other sections. Each section contains a current selection which is the last page (or section) visited by the user. Sections allow you to group related information in a hierarchical format. Each section contains a tab which provides the user access to the section.

Tab
Visible extension to a page or section that allows the user direct access to that page or section. Tabs for a notebook are contained in tab sets.

Tab Set
Holds, displays, and manages a collection of tabs. The current state and selection of the notebook is reflected in the tab set. A major and minor tab set is automatically created, if necessary, with each notebook.
