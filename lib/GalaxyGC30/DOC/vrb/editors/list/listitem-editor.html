List Item

List Programmer's Guide chapter
Use the List Item Editor to edit a list item for your application. A list item is a list view subclass provided as a convenience for implementing a one-column list of strings. For more information on list items and implementing more complex lists, refer to the List chapter of the Programmer's Guide.

Open the List Item Editor by double-clicking on a list item that has been placed in the Dialog Editor, or by selecting a list item in the Application Objects List of the Object Editor.

List Item Editor Map

Editing List Items

Specify the number of entries in the list item by editing the Row Count property in the Properties List. After specifying the row count, place-holders for each row appear in the list editor.

To edit a string in the list, first select the row and then edit the text in the text entry box below the list. Pressing Enter or changing the focus places the edited text into the list.
