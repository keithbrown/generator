Palette Item
Use the Palette Item Editor to edit a PaletteItem for your application. A palette item is a ListView subclass that displays a list of images. Palette items are often used to implement toolbars or drag-off palettes.

Open the Palette Item Editor by double-clicking on a palette item placed in the Dialog Editor or by selecting a palette item in the Application Objects List in the Object Editor.

The Palette Item Editor contains a list of images (PaletteItemElements) for the palette.

Specify the number of columns for the palette item using the Column property. The palette item automatically configures itself according to number of elements in the palette item.

To add additional images to the palette, use the Palette:New Element menu option.

To edit an image, select the image in the Image List, then edit the PaletteItemElement properties in the Property List.

Normally, you enable the Image Property and select the Open (...) button to open the Image Editor for the for the image.

You can also enable or disable the PaletteItemElement. Disabling the PaletteItemElement affects the way it is drawn.
