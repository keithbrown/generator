Combo Programmer's Guide chapter
A popdown combo is a combo subclass that enables a user to pick a selection from a list of choices. It differs from a combo in that the user can "popdown" the list of choices, rather than displaying the all of the list, or a partial list, all the time. Only the current choice is displayed in a popdown combo.

Open the Popdown Combo Editor by clicking on a popdown combo in a project, or by double-clicking on a popdown combo added to a dialog in the Dialog Editor.

The Popdown Combo Editor is functionally identical to the Combo Editor. Refer to it for details on editing popdown combos.
