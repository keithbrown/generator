Font Chooser Programmer's Guide chapter
Use the Font Chooser to specify fonts and typefaces. Open the Font Chooser from the Properties list in the Object Editor.

Font Chooser Map

  • Font Display Area
    Displays an example of the currently selected font.

  • Typeface Family List
    Lists the typefaces available on your system.

  • Face List
    Lists the faces available for the font family selected in the Family List.

  • Size List
    Lists the sizes available for the currently selected font family. You can specify additional sizes in the text field above the list. New sizes are added to the list.

Specifying Fonts and Typefaces

To specify a font and typeface, select a typeface family, face, and size.
