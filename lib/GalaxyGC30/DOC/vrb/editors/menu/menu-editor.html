Menubar Editor
Use the Menu Editor to add menu items to a menu and define their behavior.

Menus are usually built upon a menubar. However, a pop-up menu can be attached to any dialog item (using its Menu property). A pop-up menu is opened with the menu button of the mouse. Additionally, an option menu is a specially defined pop-up menu that provides a convenient way to place a menu within a dialog.

You can open the Menu Editor by clicking on a menu object in the Application Objects List, or from the Menu Property in the Property List for any item containing a Menu property.

Use the Menu Editor to:

  • Add menu items.
  • Remove menu items.
  • Rearrange the order of menu items.
  • Define or remove accelerator keys for a menu item.
  • Specify mnemonic keys for a menu item.
  • Enable or disable menus.

Adding a New Menu Item

You can add the following menu items to a menu:
  • Menu item
  • Submenu item
  • Separator
  • Toggle menu item
To add menu items, do either of the following:

  • Select from the available choice in the Menu Menu.

  • Select the Menu page of the Components Notebook. Drag an icon for the appropriate menu item onto the Menu Element List in the Menu Editor.

Removing a Menu Item

To remove a menu item, select Clear or Cut from either the Edit menu or the Editing Toolbar.

Rearranging Menu Items

To change the order of menu items on a menu do either of the following:

  • Use the Cut, Copy, or Paste commands from the Edit menu or the Editing Toolbar.

  • Select a menu item in the Menu Element List, and drag it to a new location.

Defining Accelerator Keys

An accelerator key allows your users to access a menu from the keyboard. Accelerator keys must be unique among all menu options.

To set an accelerator key for a menu item:

  1. Select the menu item in the Menu Element List.
  2. In the Property page for the selected menu item, select the Accelerator property.
  3. Type the key sequence for the Accelerator key. For example, hold down the Control key and type K to get Ctrl+K.
To disable an accelerator key, replace the key sequence with '0' (the numeral zero, not the letter 'O') or select Menu:Disable Accelerator.

Setting Mnemonic Keys

A mnemonic key allows your users to open a menu or invoke a menu command by pressing the specified key in combination with the mnemonic key modifier (usually the Alt key). If the specified key is in the title of the menu or menu item, the corresponding letter is underlined on the display.

Only one alphanumeric character can be specified as the mnemonic key. Mnemonic keys must be unique. Mnemonic keys are not case-sensitive; the upper case and lower case version of a character are not unique mnemonic key specifications.

To set a mnemonic key for a menu item:

  1. Select the menu item in the Menu Element List.
  2. In the Property page for the selected menu item, select the Mnemonic property.
  3. Type the character for the mnemonic key.

Marking a Menu Item Disabled

Disabled menu items appear dimmed and do not respond to events.

To specify a menu item as disabled, select the menu item. In the Property page for the menu item, set the Enabled property to False.
