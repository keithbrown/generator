Text item
Use the Text Item Editor to define the appearance of text in a text item.

From the Text Item Editor, you can do the following:

  • Add or edit text.
  • Format text in the text field.
    • Select text for editing and formatting.
    • Set text margins.
    • Specify justification.
    • Set tabs.
    • Set line spacing.
You can also edit text properties on a text item's Properties page to further specify the text item's behavior.

Adding Text

To enter text for a text item, click in the text editing area and begin typing.

Editing Text

To edit text, use the Cut, Copy, Paste, or Clear commands from the Edit menu or the Toolbar.

Selecting Text

Do any of the following to select text in the text editing area:

  • Drag the mouse over the text.

  • Place the cursor in the text, then use the arrow keys from the keyboard while holding down the shift button.

  • To select a word, double-click on the word.

  • To select a line, triple-click on the line.

Setting Margins

Use the text margin and indent icons on the text ruler to set margins. For these features to work properly, the Wrap property for the text item must be set to False.

  • The left margin icon sets the left margin.

  • The first line indent icon sets the indentation for the first line of the paragraph.

  • The right margin icon sets the right margin.

    The Text Item editor, by default, sets the right margin to the edge of the actual text item. If you move the right margin beyond the right edge of the text item, the right margin will still default to the right edge of the text item.

To set margins for one or more paragraphs:

  1. Click in a paragraph (or select several paragraphs).
  2. Select an indent icon on the text ruler.

Specifying Justification

Use the justification icons below the text ruler to align blocks of text.

To specify justification for one or more paragraphs:

  1. Click in a paragraph (or select several paragraphs).
  2. Select a justification icon.

Setting Tabs

Text items, by default, specify regular tab stops, as indicated on the text ruler in the Text Editor. You can use the use the tab icons below the ruler to set additional tab stops.

The left tab icon aligns the left-most character of text to where the tab stop is set. The center tab icon aligns the center of the selected text to where the tab stop is set. The right tab icon aligns the right-most character of text to where the tab stop is set. The decimal tab icon aligns decimal points to where the tab stop is set. This is useful when aligning columns of numbers.

Note that for text items to be able to accept tab characters, the Accept Tab property must be set to True. Otherwise, pressing the tab character causes the item to lose focus in the dialog.

To specify additional tab settings for one or more paragraphs:

  1. Click in a paragraph (or select several paragraphs).
  2. Select a tab icon and drag it onto the ruler.

Setting Line Spacing

Use the line spacing icons below the text ruler to set line spacing. Each icon represents single space, double space, or triple space.

To specify line spacing for one or more paragraphs:

  1. Click in a paragraph (or select several paragraphs).
  2. Select a line spacing icon.
