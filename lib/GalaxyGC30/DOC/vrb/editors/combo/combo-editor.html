Combo Programmer's Guide chapter
A combo is a dialog item subclass that enables a user to pick a selection from a list of choices.

Open the Combo Editor by clicking on a combo in a project, or by double-clicking on a combo added to a dialog in the Dialog Editor.

Use the Combo Editor to:

  • Add choices to a combo
  • Remove choices from an Combo
  • Arrange the ordering of choices within an combo

Adding Choices to an Combo

To add a new choice to a combo, select the Combo:New Choice menu option. A new choice with the tag "New Value" and the title "New Value" is added to the combo.

Removing Items from an Combo

To remove choices from an combo, select the choice you want to remove. Then, use Clear or Cut from the Edit menu or the toolbar.

Arranging Choices Within an Combo

You can control the order in which the choices are displayed in a combo. Select the choice you want to move, and then drag it to its new position in the list of choices.
