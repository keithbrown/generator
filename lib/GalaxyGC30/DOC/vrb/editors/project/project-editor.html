Preferences


The Project Window is the initial window that opens when you start the Visual Resource Builder, and serves as the "main" window. There is always a single Project Window open when the Visual Resource Builder is running. Closing the Project Window quits the Visual Resource Builder.

Use the Project Window to do the following:

  • Create new resource files.
  • Edit resource files by launching editors

Project Window Map

The Project Window contains a list of open resources.

The Project List

Each entry in the Project List displays an icon and the name of the resource file. Two small images located to the right of the resource's icon are used to represent different states the component is in:

  • A blue X indicates that a resource file has been modified but not saved.

  • A red circle indicates that the resource file has been modified since the last time the project was built.

Adding Components to a Project

To add a new resource file, choose from the File:New menu. When a new resource is added to a project, the Object Editor is opened automatically. You can disable this behavior through the preferences.

By default, new resource files are created in the same directory as the Visual Resource Builder. You can modify the default directory by setting the preferences.

Editing Components

The Object Editor opens automatically when you add new resource files. You can also open the Object Editor by double-clicking on a resource.
