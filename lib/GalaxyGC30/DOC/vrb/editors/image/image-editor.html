Image Item
Use the Image Editor to create and edit images in various image formats.

Images can be attached to any dialog item by setting the item's Image property to Custom Image. When the Image property is set, select the ellipsis button to open the Image Editor. The image you create, edit, or import into the Image Editor becomes the item's image. From the Image Editor you can do the following:

  • Import and export images.
  • Select editing tools from the Tool Palette.
  • Set the size of an image.
  • Change the view and other drawing defaults.
  • Set drawing colors.
  • Specify line widths for drawing.
  • Select areas of an image for editing.
  • Perform editing operations on selected areas.
  • Add text to an image.
  • Undo or erase editing changes to an image.
  • Install or edit color palettes for an image.
  • Create and edit image transparencies.

Image Editor Map

The Image Editor contains the following:

  • Image Editing Area
    "Canvas" area for the image that you are editing. Use scroll bars or the Hand Tool to pan through oversize images.

    When first opening an image, the Image Editing Area displays a zoomed (Fat Bits) view of the image. To edit the image in its actual size, unselect Fat Bits from the View menu.

  • Tool Palette
    Contains drawing and selection tools for editing images. When you select a tool from the Tool Palette, the cursor changes to reflect the tool you are using.

  • Foreground Color Box
    Displays the currently selected foreground color for drawing.

    To select a new foreground color, click and hold the Foreground Color Box. The Color Palette is displayed, with the current foreground color indicated on the palette. While the Color Palette is displayed, drag the pointer to a new foreground color.

  • Background Color Box
    Displays the currently selected background color for drawing.

    To select a new background color, click and hold the Background Color Box. The Color Palette is displayed, with the current background color indicated on the palette. While the Color Palette is displayed, drag the pointer to a new background color.

  • Color Palette
    The color palette installed with the image. This palette contains up to 256 colors and is not normally visible when editing an image. You can view the Color Palette by clicking and holding the mouse on either the Foreground or Background Color Box.

    Use the Color Palette Editor to view or edit colors on the Color Palette or to install different color palettes.

  • Recently Used Colors
    Palette containing the 16 most recently used colors in your image. When you first load an image into the Image Editor, this palette contains the first 16 colors from the Color Palette for the image. As you select new colors for editing, they are added to the Recently Used Colors Palette.

    To select a foreground color from the Recently Used Colors Palette, click on the color with the left mouse button. To select a background color, click with the right mouse button.

  • Line Width Palette
    Contains a selection of line widths you can set for the following drawing tools:

    • Line Tool
    • Ellipse Tools
    • Rectangle Tools

    To set the line width, select one of the four widths available.

  • Transparency Editing Area
    "Canvas" area to create transparencies for an image. Use transparencies to mask an image. Areas that are not masked are transparent; background colors in the application are visible in transparent areas.

Importing and Exporting Images

To import an image:

  1. From the File Menu, select Import...
  2. Use the Filters option in the file chooser to specify an image format.
  3. Select the file you want to import, and apply your selection in the file chooser.

To export an image:

  1. From the File Menu, select Export...
  2. Images can be exported in various image formats. Use the Filters option in the file chooser to specify an image format for the exported file.
  3. Specify the path and file name for your exported file in the file chooser and apply your selection.

Setting the size of an Image

Select Image Menu:Set Size, and type in a new height and width. Specify the height and width in pixels.

Changing the View of an Image

The following options are available from the View menu.

  • Fat Bits
    Magnifies the image so each pixel can be viewed. When Fat Bits is disabled, the image returns to its actual size. (The default magnification is five times the size of the image.)

  • Draw From Center
    Defines the starting point for certain drawing tools. Normally, the line, ellipse, and rectangle tools begin drawing an item from the end (or perimeter) of the item. Draw From Center specifies that drawing begins at the center of the item.

  • Show Grid
    Displays the image in a grid. Individual pixels are outlined with the grid pattern. When the bit size displayed is smaller than three pixels, the grid is not shown.

  • Show Position
    Displays the coordinates of an image. Note that the origin of the image is the lower left corner.
    • X: The position of the item on the X axis
    • Y: The position of the item on the Y axis
    • DX: The change in X as you draw or move an item
    • DY: The change in Y as you draw or move an item
    • Len: The distance between the starting point of the operation and the current position
    • Ang: The arc, in degrees, described from the starting point as you draw or move an item

  • Preview
    Displays the entire image in a separate Preview Window. If you use the Hand Tool to pan through an oversize image, the visible area of the image is outlined in the Preview Window.

  • Zoom In
    Magnifies the image. Select repeatedly to continually increase the magnification.

  • Zoom Out
    Decreases the magnification of the image. Select repeatedly to continually decrease the magnification.

Setting Colors for Drawing

The following tools from the Tool Palette use foreground and background colors for drawing:

  • Spray Can Tool
  • Pencil Tool
  • Line Tool
  • Ellipse Tools
  • Rectangle
  • Tools
Use any of the following methods to set the foreground and background colors for drawing.

To set the foreground color using the Color Palette:

Click on a color in the Color Palette.

The color is displayed in the Foreground Color Box. Colors can be set before or after clicking on a drawing tool.

To set the background color using the Color Palette:

Press the Control key and click on a color in the Color Palette.

The color is displayed in the Background Color Box. Colors can be set before or after clicking on a drawing tool.

To set the foreground color using the Color Palette:

  1. Click and hold the Foreground Color Box.
    The Color Palette is displayed. The current foreground color is indicated in the palette.
  2. Drag the pointer to a color in the Color Palette and release the pointer.
    The foreground color is set to the color you selected. The color is displayed in the Foreground Color Box and is added to the Color Palette.
Similarly, the background color can be set from the Background Color Box.

To set the foreground color using the Dropper Tool:

  1. Select the Dropper Tool from the Tool Palette.
    The cursor changes to an eye-dropper.
  2. With the Dropper Tool, select a color in the Image Editing Area, or from the Color Palette.
    The foreground color is set to the color of the pixel you selected and is displayed in the Foreground Color Box.
To set the background color, press the Control key while selecting a color with the Dropper Tool.

Line Width Palette

Use the Line Width Palette to set the line width for drawing. Setting the line width applies to the following drawing tools from the Tool Palette:
  • Line Tool
  • Ellipse Tools
  • Rectangle Tools
To set the line width, click on one of the four widths available.

Selecting Areas of an Image for Editing

Use the Selection Tool and the Lasso Tool to manipulate whole images or parts of images. The orientation of selected areas can be changed with editing commands from the Edit menu.

Moving or copying a selected area

Select an area of an image with the Selection Tool or the Lasso Tool. Click and drag the selected area to a new position. Press the Control key to copy, not move, the area.

Or:

Use the Cut, Copy, and Paste commands from the Edit menu.

Changing the orientation of a selected area

Select an area of an image with the Selection Tool or the Lasso Tool. From the Edit menu, select:

  • Rotate to rotate the selection 90 degrees
  • Flip Horizontal to mirror the selected area horizontally
  • Flip Vertically to mirror the selected area vertically

Cropping an image

You can crop the entire image to a selected area. The remaining unselected area of the image is discarded.

Select an area of an image with the Selection Tool or the Lasso Tool. Then select Edit:Crop To Selection.

Adding Text to an Image

Use the Text Tool to add text to an image. Select the foreground and background colors for text as you would for a drawing tool. The font and other attributes of the text should be specified before adding text to an image. Select Options:Choose Font to edit the font, size, and attributes.

To add text to an image, select the Text Tool and click in the image. An Image Text Window appears. Enter text in the Image Text Window, and click OK.

Using Undo and Erasing

There are several ways to undo changes to an image you are editing.
  • Use the Eraser Tool to erase large areas of an image.

  • Use thePencil Tool to erase areas pixel by pixel.

  • Select Edit:Undo to reverse your last editing change. Select Undo repeatedly to undo multiple changes.

  • To completely clear the Image Editing Area to the background color, click in the Image Editing Area. Then double-click the Eraser Tool.
