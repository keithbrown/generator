Project Window
Use the Preferences Editor to customize the development environment. Open the Preferences Editor by selecting the Edit:Preferences... menu choice in the Project Window.

Preferences Editor Map

Preference List

All available preferences are shown in the preferences list. Preferences are grouped by topic. To see all of the items in a topic, set the arrow to point down. To compress the preferences for a topic so only the topic name is shown, set the arrow so it points to the right. Toggle the arrow by clicking on it.

Select a preference by clicking on it in the Preferences List. A preference must be selected before it can be edited.
