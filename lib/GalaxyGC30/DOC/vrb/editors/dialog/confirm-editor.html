Use the Confirm Editor to create confirm dialogs for your application. A confirm dialog enables a user to confirm a specific action before an application continues to execute. Various confirmation buttons are used to define the available actions for the user in a confirm dialog. The image below shows an example of a typical confirmation dialog.

A confirm dialog is a dialog subclass; the Confirm Editor is a variation of the Dialog Editor.
