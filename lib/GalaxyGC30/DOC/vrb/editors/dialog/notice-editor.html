Notice Dialog
Use the Notice Editor to create notice dialogs for your application. A notice dialog communicates information to a user, usually to inform the user of an error or some irrevocable action that has been initiated. By default, notice dialogs are modal; the user must respond to the notice before proceeding in an application. Specifying Notification Text

Type the text that you want displayed in your notice in the "Notice Text:" field of the Notice Editor.

Notice Dialog Types

A notice dialog comes in two different varieties, warning notice and a caution notice. You can specify what type the notice dialog should be by modifying the "Notice Type" property.

Warning Notice

The warning notice is typically used to display information. The warning notice usually contains the following, depending on your platform's look and feel:

  • Text for the notice
  • OK button
  • Cancel button (optional)
The optional Cancel button is set using the "Has Cancel" property (the default is False).

Caution Notice

The caution notice is typically used when the user has a choice of actions or can cancel the current operation. The caution notice usually contains the following, depending on your platform's look and feel:

  • Text for the notice
  • Yes button
  • No button
  • Cancel button (optional)
The optional Cancel button is set using the "Has Cancel" property (the default is False).
