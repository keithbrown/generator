Color Chooser Programmer's Guide chapter
Use the Color Chooser to specify colors for your Galaxy applications. The Color Chooser allows you to specify colors from four different color models.

If your platform does not have a native color chooser, Galaxy uses its own. The following documentation applies only to the Galaxy color chooser. For help on native color chooser, refer to your platform's documentation.

From the Color Chooser you can do the following:

  • Select a view in the color chooser
  • Select a color model
  • Specify a color
  • Revert to a previously specified color
  • Specify color tolerances for dithering
  • Save color specifications for future use.

Color Chooser Map

  • Color View Area
    Displays tools for selecting and defining colors. Three different views can be specified using the Views menu:

    • Color Wheel or Plane
    • Named Colors
    • Color Tolerances

  • Color Method Option Menu
    Selects the color model to use when specifying a color. Four color models are available:

    • HLS
    • RGB
    • CMY
    • Grayscale

  • Color Box
    Shows the current color selection in the upper portion; the color you are editing in the lower portion.

  • Color Sliders
    Specifies the values for each component of a color. The component specified by each slider varies according to the color view model.

  • Saved Colors Palette
    Saves previously selected colors for future use. Up to 16 colors can be saved.

Specifying a View in the Color Chooser

To specify a view for the Color View Area, select from the Views menu. The default view is the HLS Color Wheel.

The Color View Area can display any of the following:

  • Color Wheel or Color Plane
    When specifying colors from the HLS color model, a color wheel is displayed. When specifying colors from the RGB or HLS color model, a color plane is displayed.

  • Named Colors
    A scrolling list of predefined colors. You can select a named color directly from the list.

  • Color Tolerances
    A set of tools to define the appearance of colors on a display by specifying a dithering method. Dithering is the process of approximating a color not directly available on a display by combining available colors in a pattern.

Specifying Colors

The Color Wheel (or Color Plane) displays a graphic representation, or map, of the selected color model. The cross-hair in the map indicates the currently specified color.

To specify a color in the Color Chooser, do any of the following:

  • HLS Color Model
    In the Color Wheel, specify a Lightness value using the Light slider. Then choose a color directly from the map by clicking on it. After clicking on the map, you can also drag the cross-hair to new positions to change the selection.

  • RGB and CMY Color Models
    From the Options menu select a color plane to display. Then choose a color directly from the map by clicking on it. After clicking on the map, you can also drag the cross-hair to new positions to change the selection.

  • All Color Models
    Use the color sliders to specify the components of a color. You can drag a slider's indicator to a new position, click on the color you want in the slider, or specify a value for the component in the adjacent text field.

    By default, the sliders dynamically update to reflect your selection. However, you can set the sliders to show an unchanging scale of values for each component by selecting Options:Reform Slider Images from the Color Chooser menu.

Named Colors

You can also select a color from a series of predefined colors. To select a predefined color, specify Named Colors from the Views Menu. Then select a color from one of those displayed in the list.

Reverting to a Previously Selected Color

Use the Color Box to revert to a previously selected color. The upper portion of the Color Box shows the current color selection; the lower portion shows the color you are editing. Revert the current color selection to the previous selection by clicking in the upper portion of the Color Box.

Color Models

The Color Chooser allows you to specify colors from the following four color models:

  • HLS
    Specifies a color using a combination of the color's hue, lightness, and saturation components. HLS is the default color model.

  • RGB
    Defines colors by specifying a mixture of the additive primary colors: red, green, and blue.

  • CMY
    Defines colors by specifying a mixture of the subtractive primary colors: cyan, magenta, and yellow.

  • Grayscale
    Defines colors as shades of gray. Grayscale is useful for defining colors for display on monochrome monitors.

Selecting a Color Model

Use the Color Method Option Menu to select a color model.

Specifying Color Tolerances for Dithering

Since many displays cannot accurately represent all colors, dithering is used to approximate colors on a display. Dithering takes place within certain tolerances you can define in the Color Chooser.

Define the color tolerances for dithering by specifying the maximum area (in pixels) to use to approximate a color.

For most applications, the color tolerances available in the Default Color Tolerances Menu are sufficient for defining colors. The options from this menu are listed below.

Menu Selection Dither Area (pixels)
Large Dither 8 x 8
Small Dither 2 x 2
Solid No Dithering. On most displays, selecting too many colors that are not dithered limits the number of colors available for other uses.
Precise Large Dither 16 x 16
Precise Small Dither 4 x 4
Precise Solid No Dithering. On most displays, selecting too many colors that are not dithered limits the number of colors available for other uses.

If you require more precision in defining a color, specify Color Tolerances View from the Views menu. This view allows you to specify a delta value that represents how close a color must match before dithering approximates the color.

Saved Colors

Use the Saved Colors Palette to save frequently used colors. Up to 16 colors can be saved. When a color is saved, the color tolerance values for dithering are saved with it. Saved colors are automatically saved with the Color Chooser and are available for future editing sessions.

Saving Colors

There are two ways to save a color on the Saved Color Palette.

To save a color by clicking on a cell:

  1. Select a color in the Color Chooser.
  2. Press the Control key and click a cell on the Saved Color Palette.
To save a color using the arrow keys:

  1. Select a color in the Color Chooser.
  2. Click on the Saved Colors Palette.
  3. Use the left or right arrow keys to move the cell indicator to select a cell for the saved color.
  4. Select Add Saved Color from the Options menu.

Retrieving Saved Colors

To retrieve a color from the Saved Colors Palette, double-click on a cell in the Saved Colors Palette.
