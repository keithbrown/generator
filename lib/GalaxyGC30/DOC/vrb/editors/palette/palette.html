Palette Editor

Palette Programmer's Guide chapter
A palette is a listview subclass that displays icons representing an object or action available to a user.

Open the Palette Editor by clicking on a palette in a resource, or by double-clicking on a palette added to a dialog in the Dialog Editor.

Use the Palette Editor to:

  • Add items to a palette
  • Set an image for the palette item
  • Remove items from an palette
  • Arrange the ordering of items on a palette

Adding items to a Palette

To add a new item to a palette, select the Palette:New Element menu option. A new choice with the tag "New Element" and an empty title is added to the palette.

Setting an Image for a Palette Item

In the Properties List, find the "Image" property. Make sure its checkbox is checked, and click the ellipsis (...) button. An Image Editor will open.

Removing Items from a Palette

To remove choices from an palette, select the item you want to remove. Then, use Clear or Cut from the Edit menu or the toolbar.

Arranging Choices on a Palette

You can control the order in which the choices are displayed on a palette. Select the item you want to move, and then drag it to a higher or lower position on the palette.
