Container Programmer's Guide chapter
A container is a dialog item that holds other dialog items. It is often used to group dialog items that behave as a unit.

Open the Container Editor by double-clicking on a container item placed in a dialog in the Dialog Editor, or by selecting a container object added to the Application Objects List in the Object Editor.

The Container Editor is a variation of the Dialog Editor. For details on using the Container Editor, refer to similar features described for the Dialog Editor.

Use the Container Editor to:

  • Add items to a container
  • Position items in a container
  • Remove items from a container
  • Arrange items within a container
  • Resize a container
  • Open dialog item editors
Note that you can also use the Dialog Editor to place items into or remove items from a container.

Container Editor Map

Adding Items to a Container

Adding a New Item

To add a new item to a container, select the Dialog Items page of the Classes Palette. Then, click and drag the item from the Classes Palette into the Editor Frame.

Positioning Items in a Container

To position items in the Container Editor, use the same Dialog Editor procedures for positioning items.

Removing Items from a Container

To remove dialog items from a container, select the items you want to remove. Then, use Clear or Cut from the Edit menu or the toolbar, or press the Delete key from the keyboard.

Arranging Items Within a Container

Like a dialog, a container contains a spring solution that defines the relative placement of its dialog items. To edit a container's spring solution, use the same Dialog Editor procedures for editing spring solutions.

Resizing a Container

To resize a container, click on any corner of the Container Frame and drag it to the desired size.

If you resize a container beyond the left or top margin of the Editor Stack, the Container Frame redraws itself within the Editor Stack. To view part of a dialog that is beyond the right or bottom side of the Editor Stack, either resize the window or allocate more space to the Editor Stack by using the window pane sliders.

You can also resize a container by editing its Rect property. Refer to dialog item properties for more information.

Opening Dialog Item Editors

To open an editor for a dialog item placed in the Container Editor, double-click on the item or select the item and press Return or Enter. An editor for the item opens and is placed on top of the Editor Stack.

After editing an item, return to the Container Editor by popping the editor for the dialog item off the Editor Stack.
