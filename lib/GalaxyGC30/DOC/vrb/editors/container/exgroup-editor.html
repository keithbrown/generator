An exclusive group is a container that holds a series of controls where only one control can be selected at a time.

Open the Exclusive Group Editor by double-clicking on an exclusive group in a project, or by double-clicking on an exclusive group added to a dialog in the Dialog Editor.

Use the Exclusive Group Editor to:

  • Add choices to an exclusive group
  • Remove choices from an exclusive group
  • Arrange the ordering of choices within an exclusive group

Adding Choices to an Exclusive Group

Adding a New Control

To add a new control to a exclusive group, select the Exclusive Group:New Choice menu option. A new choice with the tag "NewBtn" and the title "Choice" is added to the exclusive group.

Removing Items from an Exclusive Group

To remove choices from an exclusive group, select the choice you want to remove. Then, use Clear or Cut from the Edit menu or the toolbar.

Arranging Items Within an Exclusive Group

You can control the order in which the choices are displayed in an exclusive group. Select the item you want to move, and then drag it to its new position in the list of choices.
