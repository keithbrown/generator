Format Programmer's Guide chapter
A format is a pattern that represents various data types such as dates and boolean values. The format can be used in your application to accept input for the data type or define how the data type is displayed or printed. Use a Format Editor to specify a pattern to represent a data type.

Formats are based on the format types used in Microsoft Excel.

You can create a format by dragging the format to the Application Objects List. The following Format Editors are available from the Formats Page of the Object Notebook, which is located in the Object Editor.

  • Boolean
    Specifies how to represent a true or false value.

  • Date
    Specifies how to represent a date value.

  • Integer
    Specifies how to represent an integer value.

  • Double
    Specifies how to represent a double value.

The Format Editor contains the following panels:

  • Sample
    Displays an example of the specified format.

  • Patterns
    Displays a list of patterns that can be used to represent a database type.

    Add new patterns to the list by selecting a pattern in the Sample Patterns List, and then selecting the Add button or the Format:Add Format menu option.

    Remove a pattern by selecting the pattern in the Pattern List and then selecting the Remove button or the Format:Remove Format menu option.

    Create a new pattern by editing an existing pattern in the list. After selecting a pattern in the list, select the pattern again to place the pattern in a one line text field. Edit the pattern in the text field to create the new pattern.

  • Format Editor Buttons
    • Add
      Adds the selected pattern from the Sample Patterns List to the Pattern List.

    • Remove
      Removes the selected pattern from the Pattern List.

    • Set Output
      Specifies that the selected pattern in the Pattern List is the default pattern for that database type.

  • Pattern Templates
    Lists the sample patterns available for the database type. To add a pattern, select a sample pattern and then select the Add button or the Format:Add Format menu option.
