Menu Editor







A menubar is a dialog item subclass that displays an ordered list of menus. Menubars usually host the top level menus for an application.

Open the Menubar Editor by double-clicking on a menubar item added to a dialog in the Dialog Editor, or by setting a custom menu in the Properties List for a dialog. The Dialog Editor and Properties List are part of the Object Editor.

Use the Menubar Editor to:

  • Add menus to a menubar.
  • Remove menus from a menubar.
  • Change the order of menus within a menubar.
  • Edit menu properties.

Adding Menus to a Menubar

You can add menus (and menu elements to a menubar using the Object Editor's Menubar Menu, or you can drag a menu (or menu element) from the Menu Page of the Components Notebook into the Menubar Editor. Removing a Menu from a Menubar

To remove a menu from a menubar select the menu in the Menu List. Then select Clear or Cut from either the Edit menu or from the Toolbar.

Rearranging Menus

To change the order of menus on a menubar do either of the following:

  • Use the Cut, Copy, or Paste commands from the Edit menu.

  • Select a menu in the Menu List, and drag it to a new location.

Editing Menu Properties

The Title property defines the name for a menu. To edit the name of a menu:
  1. Select the menu in the Menu List.
  2. In the Properties page for the selected menu, select the Title property.
  3. In the text field for the Title property, type in the name of the menu.
The Tag property provides access to the menu in your application code. To edit the tag of a menu:
  1. Select the menu in the Menu List.
  2. In the Properties page for the selected menu, select the Tag property.
  3. In the text field for the Tag property, type in the name of the menu.
