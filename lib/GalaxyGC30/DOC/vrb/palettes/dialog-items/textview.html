Text chapter
vtextview Prefix
A textview (TextView) is a graphic object that is often used in association with a TextData object. The separation between the data object and graphical object allows you, as a programmer, to manipulate either object without necessarily having the overhead involved with both objects at the same time.

For example, you might want to sort the text. Since this requires some time to perform, you can disassociate the textview from the text data, sort the data, and then reassociate the textview with the text data again. This way, the graphical interface isn't always trying to keep up with the text sorting routines you might be performing.

The textview, therefore, serves as the graphical front end of a text data object that is programmatically associated with it.

For more information on textviews, refer to the following:

  • Text chapter in the Programmer's Guide.
  • vtextview class in the Reference Manual.

Key Properties

Property Defaults Description
Auto Scroll True Designates whether a textview auto-scrolls when the selection is being tracked.
Drag-Drop Edit False Enables moving text by selecting the text, and dragging it to new locations.
Grabbing Hand False Determines whether the "grabbing hand" (used to "grab" the screen for scrolling purposes) is enabled for this textview.
Highlight Style Default Defines the appearance of highlighted text. Highlighted text is text that is selected for editing.

The possible values include:

  • Box
  • Default for this platform
  • Dotted Line
  • Invert
  • Line
  • None
Horizontal Scroll Bar False Determines whether the textview has a horizontal scroll bar.
Horizontally Locked False Determines whether the textview allows its text to be horizontally scrolled.
Left Margin 10 Sets the number of pixels between the left edge of the text field border and the first character.
Scroll Bar Margin 2 Sets the number of pixels between the edge of the scroll bar and the text in the text field.
Tag TextView Tag name of the textview, for referencing in application code.
Title Nothing The title of the textview, often not displayed.
Top Margin 3 Sets the number of pixels between the text field border and the top of the first line of text.
Unhighlight Style Default Defines the appearance of unhighlighted text. Unhighlighted text is selected text in a window that no longer has the focus.

The possible values include:

  • Box
  • Default for this platform
  • Dotted Line
  • Invert
  • Line
  • None
Vertical Scroll Bar False Determines whether the textview has a vertical scroll bar.
Vertically Locked False Determines whether the textview allows its text to be vertically scrolled.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
