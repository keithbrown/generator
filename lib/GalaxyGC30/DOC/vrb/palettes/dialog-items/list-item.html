List chapter
vlistitem Prefix
A list item (ListItem is a limited listview subclass that provides a convenient display of a single column list of strings. When a listitem is created, an underlying list data object is created with it.

The list item provides convenient methods to access the elements in the list, append or prepend elements, or remove elements from the list item. List items are useful when presenting simple choices to an end-user, such as the list of files in the File Chooser, or the list of fonts in the Font Chooser.

More complex lists that require multi-dimensional display of data (more than one column) should use the listview class.

For more information on list items, refer to the following:

  • List chapter in the Programmer's Guide.
  • vlistitem class in the Reference Manual.

Key Properties

Property Defaults Description
At Least One Selection False Specifies that at least one item in the list is selected at all times.
Auto Scroll True Specifies whether the palette automatically scrolls when the user drags outside of the item's boundaries.
Auto-Calculate Column True Specifies that the width of a column is determined by the greater of:
  • the width of the largest item in the underlying list
  • The width of the list item
If Auto-Calculate Column is set to False, the width of a column equals the width of the list item.
Column Rearrange False Determines whether the columns of the list item can be rearranged.
Drawing Selection False Specifies the drawing selection mode for the palette item. If Drawing Selection is disabled, dragging the mouse during selection sweeps out a rectangular area, if it is enabled, dragging the mouse selects only those cells the mouse pointer passes over.
Grabbing Hand False Determines whether the "grabbing hand" (used to "grab" the screen for scrolling purposes) is enabled for this list item.
Horizontal Grid False Determines whether the horizontal grid lines are drawn for this list item.
Horizontal Scroll Bar False Determines whether the list item has a horizontal scroll bar.
Keyboard Selection True Determines whether the list item allows its items to be selected using the keyboard.
Keyboard Transversal True Determines whether the list item allows the keyboard to transverse through the list.
Natural Number of Rows Zero The number of rows used to calculate the natural height of a list item. A value of zero means all rows are used to calculate the natural height.
Rearrange False Determines whether the list item's items can be rearranged.
Row Count Zero Specifies the number of rows in a list.
Row Rearrange False Determines whether the list item's rows can be rearranged.
Selection Method One Only Selection Determines the selection method for the list item. The following values are available:
  • Free Selection
    Selecting a cell adds it to the current selection, regardless of location.
  • No Selection
    No selections are allowed.
  • One Only Selection
    Only one cell can be selected at any time.
  • Rectangular Selection
    A rectangular section of cells can be selected.
Selection Sense True Determines whether or not the list item can sense when a selection has changed.
Tag ListItem Tag name of the list item, for referencing in application code.
Title Nothing The title of the list item, often not displayed.
Traversal Wrap True If Keyboard Traversal is enabled, determines if the "focus" wraps to the next line in the list item.
Vertical Grid False Determines whether the list item displays the vertical grid lines.
Vertical Scroll Bar False Determines whether the list item has a vertical scroll bar.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
