vslider Prefix
A slider (SliderItem) is a control subclass that allows a user to specify a value by moving a "thumb" over a linear range or by clicking on the slider to "page" through the available values.

For more information on sliders, refer to the following:

  • Control chapter in the Programmer's Guide.
  • vslider class in the Reference Manual

Key Properties

Property Defaults Description
Alternate Style False Draws the slider in an alternate style under OPEN LOOK look-and-feel.
Exclusive False Inherited from the control item. Sliders should always have an exclusive False value.
Increment 1 Determines the amount the slider will increment its value.
Max 1 Determines the maximum value the slider can have.
Min 0 Determines the minimum value the slider can have.
Page 1 Base unit for increasing (or decreasing) the control value when the slider is "paged." Each time a user clicks on the slider outside the thumb, the value changes by the page value.
Tag SliderItem Tag name of the slider, for referencing in application code.
Title Nothing The title of the slider, usually not displayed.
Vertical False Specifies whether the slider should be drawn vertically or horizontally.
Value 0 Determines the initial value of the slider.
Consult the Dialog Item Key Properties table and the Control Key Properties table for additional properties that are inherited by this object.
