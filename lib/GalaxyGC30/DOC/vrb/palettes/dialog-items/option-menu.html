An option menu (OptionItem) is a control subclass that provides a pop-up menu. The value of the option menu is the index of one of the menu elements in the menu, ranging from zero to the number of elements minus one.

When the option menu is not active, either the default menu element or the last selected menu element is displayed.

For more information on option menus, refer to the following:

  • Control chapter in the Programmer's Guide.
  • vcontroloptionmenu class in the Reference Manual.

Key Properties

Property Defaults Description
Max Max Determines the maximum number of choices on the menu. Set to one less than the number of menu elements.
Min 0 Determines the minimum number of menu elements. Set to zero. If set to any other value, the menu may not display or behave properly.
Tag OptionItem Tag name of the option menu, for referencing in application code.
Title Nothing The title of the option menu, often not displayed.
Value 0 Index of the default menu element in the option menu. The first menu element is at index zero. The default menu element is initially displayed in the application.
Consult the Dialog Item Key Properties table and the Control Key Properties table for additional properties that are inherited by this object.
