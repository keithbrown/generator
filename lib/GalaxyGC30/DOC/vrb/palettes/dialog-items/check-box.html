A toggle (ToggleItem) is a control subclass with values for on and off. In some look and feels, a toggle is often referred to as a "checkbox."

For more information on toggles, refer to the following:

  • Control chapter in the Programmer's Guide.
  • vcontroltoggle class in the Reference Manual.

Key Properties

Property Defaults Description
Alternate Style False Draws the toggle in an alternate style under OPEN LOOK look-and-feel.
Exclusive False Determines if the toggle is part of an exclusive group of toggle.
Increment 1 Inherited from the control item. Toggles should never have an increment value greater than 1.
Max 1 Inherited from the control item. Toggles should never have a maximum value greater than 1.
Min 0 Inherited from the control item. On some look and feels, you can implement a "tri-state" toggle by setting Min to -1.
Page 1 Inherited from the control item. Toggles should never have a page value greater than 1.
Tag ToggleItem Tag name of the toggle, for referencing in application code.
Title Toggle The title of the toggle, displayed to the side of the toggle.
Vertical False Specifies whether the toggle should be drawn vertically or horizontally.
Value 0 1 indicates on; zero indicates off. On some look-and-feels, -1 indicates the "fuzzy" state for a "tri-state" toggle.
Consult the Dialog Item Key Properties table and the Control Key Properties table for additional properties that are inherited by this object.
