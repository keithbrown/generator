Menubar chapter
vmenubar Prefix
A menu bar (MenuBarItem) is a dialog item subclass that contains an ordered list of menus. A menu bar contains a default spring solution that places it at the top of a dialog. It also contains pre-defined File and Edit menus.

For more information on menu bars, refer to the following:

  • Menu Bar chapter in the Programmer's Guide.
  • vmenubar class in the Reference Manual.
  • Menu Bar Editor

Key Properties

Property Defaults Description
Has Help False Specifies that the last menu on the menu bar is a help menu. For some look-and-feels, the help menu is right-justified.
Is Local False Determines if the menu bar will be constrained to be local to its containing dialog.

If this property is True, the menu bar will always remain in its containing dialog regardless of the defined look-and-feel behavior.

Tag MenuBarItem Tag name of the menu bar, for referencing in application code.
Title Nothing The title of the menu bar, often not displayed.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
