A text spinner (TextSpinnerItem) is a spinner subclass that associates a list of character strings with the value of the spinner. The current value of the text spinner locates the position of a string in the list. This string is displayed in the text item. The user moves through the list of strings with the increment and decrement buttons or by typing a string directly into the text item. If the Modifiable property of a text spinner is set, then new strings typed by the user are added to the list.

Note that the text spinner displays only one string at a time. The entire list of strings for a text spinner is not displayed; the user must step through the list one string at a time to view all the available options. A text spinner is useful for lists of strings familiar to a user, such as days of the week or months of the year. If you want the user to see the available choices, use a combo.

You can programmatically add text to a text spinner by using appendText, specifying the string to insert. Remove text from the text spinner by using removeText, specifying the string to remove, or removeTextAt, specifying the index of the string in the text spinner.

For more information on text spinners, refer to the following:

  • Spinner chapter in the Programmer's Guide.
  • vspinnertext class in the Reference Manual.

Key Properties

Property Defaults Description
Alternate Style False Draws the text spinner in an alternate style under OPEN LOOK look-and-feel.
Exclusive False Inherited from the control item. Text spinners should never have this value set to True.
Increment 1 Inherited from the control item. Text spinners should never have an increment value greater than 1.
Max 1 Inherited from the control item. Text spinners should never have a maximum value greater than 1.
Min 0 Inherited from the control item. Text spinners should never have a minimum value greater than 0.
Page 1 Inherited from the control item. Text spinners should never have a page value greater than 1.
Tag TextSpinnerItem Tag name of the text spinner, for referencing in application code.
Title Nothing The title of the text spinner, often not displayed.
Value 0 1 indicates on; zero indicates off. On some look-and-feels, -1 indicates the "fuzzy" state for a "tri-state" toggle.
Vertical False Specifies whether the text spinner should be drawn vertically or horizontally.
Consult the Dialog Item Key Properties table and the Control Key Properties table for additional properties that are inherited by this object.
