Spinner chapter
vspinner Prefix
A spinner (NumericSpinnerItem) is a control subclass that allows a user to select a numeric value from a numeric range. Selections are made by incrementing or decrementing the value of the spinner, or by directly typing in the new value. If the user specifies an invalid value (for example, a value out of range or a non-numeric character), the spinner value changes to the closest matching value.

A close relative of the numeric spinner is the TextSpinnerItem, which provide a mechanism for a user to specify a choice from a list of character strings.

For more information on numeric spinners, refer to the following:

  • Spinner chapter in the Programmer's Guide.
  • vspinner class in the Reference Manual.

Key Properties

Property Defaults Description
Alternate Style False Draws the spinner in an alternate style under OPEN LOOK look-and-feel.
Exclusive False Inherited from the control item. Spinners should never have an exclusive True value.
Increment 1 Determines the amount that will be incremented or decremented from the existing value in the spinner.
Max 1 Determines the maximum numeric value for the spinner.
Min 0 Determines the minimum numeric value for the spinner. Note that this can be a negative number.
Page 1 Inherited from the control item. Spinners should never have a page value greater than 1.
Tag NumericSpinnerItem Tag name of the spinner, for referencing in application code.
Title Nothing The title of the spinner, often not displayed.
Vertical False Specifies whether the spinner should be drawn vertically or horizontally.
Value 0 Determines the "default" value of the spinner when it is first opened.
Consult the Dialog Item Key Properties table and the Control Key Properties table for additional properties that are inherited by this object.
