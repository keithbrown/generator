An exclusive group (ExclusiveGroupContainerItem) is a container subclass that allows a single selection from a group of choices.

For more information on exclusive groups, refer to the following:

  • Control chapter in the Programmer's Guide.
  • Container chapter in the Programmer's Guide.
  • vcontainerexclusivegroup class in the Reference Manual.

Key Properties

Property Defaults Description
Flat Keyboard Traversal True When True, allows the user to "tab through" the items in the group to other items in the dialog. When False, the user can "tab into" the group, but cannot "tab out."
Horizontal Scroll Bar False Specifies the presence of a horizontal scroll bar.
Tag ExclusiveGroupContainerItem Tag name of the exclusive group, for referencing in application code.
Title Exclusive Group The title of the exclusive group, displayed in the top center of the exclusive group's border.
Vertical Scroll Bar False Specifies the presence of a vertical scroll bar.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
