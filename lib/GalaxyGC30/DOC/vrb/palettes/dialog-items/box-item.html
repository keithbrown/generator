Dialog Items chapter

vdialog Prefix
A box item (BoxItem is a dialog item subclass often used as a placeholder or decoration for a dialog. Usually, you set the Border, Title, or Background Color properties of a box item. Then place other items on "top" of the box item to "logically group" the items in the dialog. Use the Send To Back menu option with a box item to ensure that other items are correctly placed on top of the box item.

For more information on buttons, refer to the following:

  • Dialog Items chapter in the Programmer's Guide.
  • vdialogboxitem class in the Reference Manual.

Key Properties

Property Defaults Description
Border True Draws a border around the box item.
Tag BoxItem Tag name of the box item, for referencing in application code.
Title Box Item Specifies the name displayed in the border of the box item.
Consult the Dialog Item Key Properties table for additional properties that are inherited by the notice dialog.
