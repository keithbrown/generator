Use the Dialog Items Palette to add dialog items to a dialog in the Dialog Editor or to a project in the Object Editor.

To add dialog items to a dialog, open the Dialog Editor by selecting a dialog (or one of its subclasses) in the Application Object List. In the Dialog Items page of the Object Palette, click and drag an icon into the Dialog Editor.

To add dialog items to a project, the Object Editor must be displayed. In the Dialog Items page of the Object Palette, click and drag an icon into the Application Objects List.

The following dialog items are available on the Dialog Items Palette:

  • BoxItem
  • ButtonItem
  • ToggleItem
  • ComboItem
  • ContainerItem
  • ControlItem
  • DialogItem
  • ExclusiveGroupContainerItem
  • ImageItem
