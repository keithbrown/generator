Button chapter
vbutton Prefix
A button (ButtonItem) is a dialog item subclass that initiates, or triggers, an action. The Title property of a button determines the name that is drawn on the button.

For more information, refer to the following:

  • Button chapter in the Programmer's Guide.
  • ButtonItem class in the Reference Manual.

Key Property

Property Defaults Description
Tag ButtonItem Tag name of the item, for referencing in application code.
Title Button Specifies the name displayed on the button.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
