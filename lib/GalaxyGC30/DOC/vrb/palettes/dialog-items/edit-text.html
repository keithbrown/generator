A text item (TextItem) is a dialog item subclass that provides an editable text field for your application. End-users can enter or edit text in editable text fields. You can also specify that the text field is static; static fields cannot be edited by the user. Static text fields are useful for presenting information to the user with various text styles applied to the text.

Note that a label item also displays static text, but has limited formatting options.

For more information on text items, refer to the following:

  • Text chapter in the Programmer's Guide.
  • vtextitem class in the Reference Manual.
Use the Text Item Editor to format and apply styles to text items.

Key Properties

Property Defaults Description
Accept Tab False Allows the text item to accept tab characters. If Accept Tabs is False, then pressing the tab character causes the text item to lose focus in the dialog.
Auto Scroll True Designates whether a text view auto-scrolls when the selection is being tracked.
Drag-Drop Edit False Enables moving text by selecting the text, and dragging it to new locations.
Grabbing Hand False Determines whether the "grabbing hand" (used to "grab" the screen for scrolling purposes) is enabled for this textview.
Highlight Style Default Defines the appearance of highlighted text. Highlighted text is text that is selected for editing.

The possible values include:

  • Box
  • Default for this platform
  • Dotted Line
  • Invert
  • Line
  • None
Horizontal Scroll Bar False Determines whether the textview has a horizontal scroll bar.
Horizontally Locked False Determines whether the textview allows its text to be horizontally scrolled.
Left Margin 2 Sets the number of pixels between the left edge of the text field border and the first character.
Modifiable True Determines whether the end-user can edit text in the text item.
One Line True Limits text to one line. If text extends beyond the line, the display of the text is truncated.
Scroll Bar Margin 2 Sets the number of pixels between the edge of the scroll bar and the text in the text field.
Selectable True Determines whether the end-user is allowed to select text.
Static False Cannot be selected or edited by end-users. Static text does not have a border, a top margin, or a horizontal scroll bar.
Tag TextItem Tag name of the text item, for referencing in application code.
Title Nothing The title of the text item, often not displayed.
Top Margin 0 Sets the number of pixels between the text field border and the top of the first line of text.
Unhighlight Style Default Defines the appearance of unhighlighted text. Unhighlighted text is selected text in a window that no longer has the focus.

The possible values include:

  • Box
  • Default for this platform
  • Dotted Line
  • Invert
  • Line
  • None
Vertical Scroll Bar False Determines whether the textview has a vertical scroll bar.
Vertically Locked False Determines whether the textview allows its text to be vertically scrolled.
Wrap True Allows text to "wrap" to the next line when the end of the current line is reached.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
