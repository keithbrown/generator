Dialog Item chapter

vdialog Prefix
A label item (LabelItem) is a dialog item subclass that displays a static text field. The Title property of a label item is the text field displayed by the label item.

Label items are usually used as identifiers in a dialog.

For more information, refer to the following:

  • Dialog Item chapter in the Programmer's Guide.
  • vdialoglabelitem class in the Reference Manual.

Key Properties

Property Defaults Description
Tag LabelItem Tag name of the label item, for referencing in application code.
Title Label Item The title of the label item, displayed as the "label" of the label item.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
