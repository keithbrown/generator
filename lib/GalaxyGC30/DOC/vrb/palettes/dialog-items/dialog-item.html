Dialog Item chapter
vdialog Prefix






In essence, a dialog item (DialogItem) is an object within a dialog that allows the user to interact with an application. The dialog item can be used in an application to specify a section of the dialog where different properties come into play. For example, a dialog item can be placed in the dialog as a backdrop with a specific background color, or perhaps assigned a particular popup menu or cursor.

You can also use the dialog item class as a starting point when creating your own custom-built dialog items.

For more information on dialog items, refer to the following:

  • Dialog Item chapter in the Programmer's Guide.
  • vdialogitem class in the Reference Manual.

Key Properties

Property Defaults Description
Abandon False Specifies that the item is an abandon item. A user activates an abandon item from the keyboard by pressing the Escape key.
Background nothing Specifies the background color of the item.
Border True Indicates whether a border is drawn around the item.
Click Focusable False Indicates whether the item is focusable using a pointing device (mouse).
Confirm False Specifies that the item is a confirm item. A user activates a confirm item from the keyboard by pressing the Enter or Return key.
Cursor nothing Indicates the default cursor when the pointer is over the item.
Default Focus Item False Specifies that the item is enabled when the dialog containing it opens. Only one item in a dialog can have default focus; setting this property automatically clears the Default Focus Item property previously set for another item in the dialog.
Enabled True Indicates if the item is active for user interaction.
Focusable False Indicates if the item can obtain keyboard focus.
Font nothing Specifies the font to use when drawing text on the item.
Foreground nothing Specifies the foreground color of the item.
Image nothing Specifies the image to be drawn on the item.
Image Mask nothing Specifies whether the image for the item contains an image mask.
Menu nothing Specifies whether the item contains a pop-up (option) menu.
Mnemonic nothing Indicates a mnemonic character for the item. A mnemonic character allows the user to activate a dialog item from the keyboard in a LAF specific fashion.
Rect 
h 100
w 100
x   0
y   0
Specifies the default size of a dialog item, and its placement in the dialog. A Rect contains the following fields:
  • h is the height of the item
  • w is the width of the item
  • x is the distance, in points, from the origin
  • y is the distance, in points, from the origin

Note that the origin is the lower left corner of the dialog (or container) containing the item.

Tag DialogItem Tag name of the item, for referencing in application code.
Title Dialog Item The title of the item, usually displayed in association with a dialog item, depending on the subclass.
Visible True Indicates whether the item is drawn in the dialog (or container).
