A container (ContainerItem) is a dialog item subclass that holds other items. It is often used to group dialog items that behave as a unit.

For more information on containers, refer to the following:

  • Container chapter in the Programmer's Guide.
  • vcontainer class in the Reference Manual.

Spring Solutions

Dialog items placed in a container have spring solutions, much like items placed in a dialog. The spring solution for items placed in a container is calculated relative to the edges of the container.

Container Editor

Use the Container Editor to to place and arrange dialog items into a container. You can also use the Edit menu in the Dialog Editor to place selected items in the dialog into a container.

Container Subclasses

The container class is a parent class to the following classes:
  • Grouped container (GroupContainerItem)
  • ExclusiveGroupContainerItem

Key Properties

Property Defaults Description
Flat Keyboard Traversal False When True, allows the user to "tab through" the items in the container to other items in the dialog. When False, the user can "tab into" the container, but cannot "tab out."
Horizontal Scroll Bar False Specifies the presence of a horizontal scroll bar.
Tag ContainerItem Tag name of the container, for referencing in application code.
Title Nothing The title of the container, often not displayed.
Vertical Scroll Bar False Specifies the presence of a vertical scroll bar.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
