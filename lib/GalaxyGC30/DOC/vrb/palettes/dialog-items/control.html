Dialog Item

Control chapter
vcontrol Prefix
A control item (ControlItem) is a dialog item subclass with a set of discrete states defined by a range of control values. Each control value within this range indicates a specific action in an application. Control values range from zero to n-1, where n is the number of values in the range.

For more information on control items, refer to the following:

  • Control chapter in the Programmer's Guide.
  • vcontrol class in the Reference Manual.

Control Subclasses

The control class is a parent class to the following classes:

  • ToggleItem
  • OptionItem
  • ScrollBarItem
  • SliderItem
  • NumericSpinnerItem
  • TextSpinnerItem

Key Properties
Property Defaults Description
Alternate Style False Draws the control in an alternate style under OPEN LOOK look-and-feel.
Exclusive False Determines if the control is part of an exclusive group of controls.
Increment 1 Sets the increment value for a control item.
Max 1 Maximum value for the control item.
Min 0 Minimum value for the control item.
Page 1 Determines the page value for a control item.
Tag ControlItem Tag name of the control, for referencing in application code.
Title Nothing The title of the control, often not displayed.
Value 0 Default placement with the range of values.
Vertical False Specifies whether the control item should be drawn vertically or horizontally.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
