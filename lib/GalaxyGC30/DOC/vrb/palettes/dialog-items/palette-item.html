Palette chapter
vpalette Prefix
A palette (PaletteItem) is a ListView subclass that displays icons representing an object or action available to a user.

For more information on palettes, refer to the following:

  • Palette chapter in the Programmer's Guide.
  • vpalette class in the Reference Manual.

Key Properties

Property Defaults Description
Cell Height 40 Specifies the default size of items on the palette.
Cell Width 40 Specifies the default width of items on palette.
Columns 2 Specifies the number of columns on the palette.
Tag PaletteItem Tag name of the palette, for referencing in application code.
Title Nothing The title of the palette, often not displayed.
Consult the Dialog Item Key Properties table and the List View Key Properties table for additional properties that are inherited by this object.
