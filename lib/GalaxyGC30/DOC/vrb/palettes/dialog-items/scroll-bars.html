A scroll bar (ScrollBarItem) is a control subclass that allows a user to specify a value by moving a "thumb" over a linear range or by incrementally specifying values with arrows at either end. The user can also click an area on the scroll bar to "page" through the available values. A scroll bar is essentially a slider with increment arrows that allow the scroll bar value to be changed by fixed amounts.

Scroll bars are usually associated with views of information such as listviews or textviews.

For more information on scroll bars, refer to the following:

  • Control chapter in the Programmer's Guide.
  • vsliderscrollbar class in the Reference Manual.

Key Properties

Property Defaults Description
Alternate Style False Draws the scroll bar in an alternate style under OPEN LOOK look-and-feel.
Exclusive False Inherited from the Control item. This should always be False.
Increment 1 The amount to increment the scroll bar when activated.
Max 1 The maximum value of the scroll bar.
Min 0 The minimum value of the scroll bar.
Page 1 The amount the scroll bar pages when the end-user clicks outside the thumb on the scroll bar.
Tag ScrollBarItem Tag name of the scroll bar, for referencing in application code.
Title Nothing The title of the scroll bar, usually not displayed.
Value 0 The default value (position) of the scroll bar.
Vertical False Specifies whether the scroll bar should be drawn vertically or horizontally.
Consult the Dialog Item Key Properties table and the Control Key Properties table for additional properties that are inherited by this object.
