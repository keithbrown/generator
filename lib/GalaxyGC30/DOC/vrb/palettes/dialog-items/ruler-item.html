Dialog Item chapter
vruler Prefix
A ruler (RulerItem) is often associated with lists, and can be used to control list column placement, sizing, and so forth.

For more information on rulers, refer to the following:

  • Dialog Item chapter in the Programmer's Guide.
  • vruleritem class in the Reference Manual.

Key Properties

Property Defaults Description
Tag RulerItem Tag name of the ruler, for referencing in application code.
Title Nothing The title of the ruler, usually not displayed.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
