The OLE palette contains a list of all of the ActiveX controls that you have installed on your system.

Note: Only those systems that support OLE and ActiveX display an OLE palette.

Dragging an ActiveX control from the palette into a dialog creates an ActiveX Control Item. An ActiveX control item is a container subclass that allows you to specify an ActiveX control type and its events and properties.

For more information on ActiveX control items, refer to the OLEControlSiteItem class in the Reference Manual.

ActiveX Control Editors

For information on editing ActiveX control items, refer to the ActiveX Control Editors.

Key Properties

Property Defaults Description
Control Type No Type Specifies the ActiveX control type. To select a control type, select the check box and click the ellipsis button to open the ActiveX Control Type Chooser.
Event Types no default value Specifies which ActiveX events are enabled for the ActiveX control Type selected. Select the Set Event button to display the available ActiveX events. Select from the available events.
Properties no default value Specifies the ActiveX properties for the ActiveX control type selected. Specify properties from the Properties page.

Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
