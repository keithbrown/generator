vlistview Prefix






A list view (ListView) is a graphic object that is often used in association with a ListData object. The separation between the data object and graphical object allows you, as a programmer, to manipulate either object without necessarily having the overhead involved with both objects at the same time.

For example, you might want to sort the list. Since this requires some time to perform, you can disassociate the list view from the list data, sort the data, and then reassociate the list view with the list data again. This way, the graphical interface isn't always trying to keep up with the list sorting routines you might be performing.

The list view, therefore, serves as the graphical front end, while a list data object is often associated with it programmatically.

For more information on list views, refer to the following:

  • List chapter in the Programmer's Guide.
  • vlistview class in the Reference Manual.

Key Properties

Property Defaults Description
Auto Scroll True Specifies whether the palette automatically scrolls when the user drags outside of the view's boundaries.
Column Rearrange False Determines whether the columns of the list view can be rearranged.
Drawing Selection False Specifies the drawing selection mode for the palette item. If Drawing Selection is disabled, dragging the mouse during selection sweeps out a rectangular area, if it is enabled, dragging the mouse selects only those cells the mouse pointer passes over.
Grabbing Hand False Determines whether the "grabbing hand" (used to "grab" the screen for scrolling purposes) is enabled for this list view.
Horizontal Grid False Determines whether the horizontal grid lines are drawn for this list view.
Horizontal Scroll Bar False Determines whether the list view has a horizontal scroll bar.
Keyboard Selection True Determines whether the list view allows its items to be selected using the keyboard.
Keyboard Transversal True Determines whether the list view allows the keyboard to transverse through the list.
Rearrange False Determines whether the list view's items can be rearranged.
Row Rearrange False Determines whether the list view's rows can be rearranged.
Selection Sense True Determines whether or not the list view can sense when a selection has changed.
Tag ListView Tag name of the list view, for referencing in application code.
Title Nothing The title of the list view, often not displayed.
Traversal Wrap True If Keyboard Traversal is enabled, determines if the "focus" wraps to the next line in the list view.
Vertical Grid False Determines whether the list view displays the vertical grid lines.
Vertical Scroll Bar False Determines whether the list view has a vertical scroll bar.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
