Notebook chapter
vnotebook Prefix






A notebook (NotebookItem) organizes and presents information in a particular style so the user can find, display, and use the information quickly and easily. Notebooks are often used in dialogs that implement preferences or other user settings.

For more information on notebook items, refer to the following:

  • Notebook chapter in the Programmer's Guide.
  • vnotebook class in the Reference Manual.

Key Properties

Property Defaults Description
Allow Partial Tabs True Determines whether partial tabs are displayed in the notebook.
Folio True Specifies whether the folio (the status bar-like rectangle, usually located at the bottom of the notebook) is displayed.
Folio Justification Left Specifies the location of the folio in the notebook. Possible values include:
  • Center
  • Left
  • Right
Major Tab Side Top Side Specifies the location of the major tabs in the notebook. Possible values include:
  • Bottom Side
  • Left Side
  • Right Side
  • Top Side
Minor Tab Side Right Side Specifies the location of the minor tabs in the notebook. Possible values include:
  • Bottom Side
  • Left Side
  • Right Side
  • Top Side
Page Borders False Specifies whether a border should be drawn around each page in the notebook.
Page Buttons True Specifies whether the buttons (next to the folio) should be drawn in the notebook.
Page Count 5 Number of virtual pages that are drawn in certain look-and-feels to help provide a visual 3D representation of a notebook.
Page Inset 1 Specifies the number of pixels that the page is inset when there are no borders are turned on for pages.
Page Inset Border 2 Specifies the number of pixels that the page is inset when there are borders are turned on for pages.
Page Step 3 The number of pixels that are in between each virtual page of a 3D representation of a notebook.
Page Tabs Major Tabs Specifies the type of tabs that are displayed in this notebook. Values include:
  • Major and minor tabs
  • Major Tabs
  • No Tabs
  • Page Tabs
Pages Wrap False Specifies whether pages should automatically wrap.
Tab Buttons Centered True Determines if the buttons to scroll the tabs in a tab set are to be centered in the tab set rectangle or aligned flush with the inside border.
Tab Buttons Placement Separate Specifies the location of the tab buttons. Possible values are:
  • Adjacent
  • Separate
Tab Style Round Rectangular Specifies the style of the tabs. Possible values are:
  • Beveled
  • Rectangular
  • Round Rectangular
Tag NotebookItem Tag name of the notebook, for referencing in application code.
Title Nothing The title of the notebook, often not displayed.
Vertical Tags Normal Determines the style of vertical tags (set by modifying the Major Tag Side property). Possible values include:
  • Normal - draw text from left to right.
  • Rotated - rotate text to parallel tab placement.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
