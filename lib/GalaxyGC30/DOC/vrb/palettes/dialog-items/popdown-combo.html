vcombobox Prefix
A popdown combo (PopdownComboItem) is a combo subclass that displays the entry box, but not the choice box, in an application. Popdown combos provide a look-and-feel defined mechanism for "popping down" (displaying) the choice box. This mechanism usually consists of selecting a button from the button box, which is located next to the entry box. After the user makes a selection from the popped down choice box, the choice box is no longer displayed.

For more information on popdown combos, refer to the following:

  • Combobox chapter in the Programmer's Guide.
  • vcomboboxpopdown class in the Reference Manual.

Key Properties

Property Defaults Description
Accept Behavior Default Determines the accept behavior of the popdown combo.

Possible values include:

  • Do Nothing
  • Match or Add
  • Match or Closest
  • Match or First
  • Match or Nothing
  • Match or Partial
  • Match or Revert
Auto-Sort False Determines whether the combo will automatically sort its contents.
Scroll Behavior Determines the scrolling behavior of the combo.

Possible values include:

  • Default
  • Do Nothing
  • Match or Add
  • Match or Closest
  • Match or First
  • Match or Nothing
  • Match or Partial
  • Match or Revert
Select Behavior Determines the scrolling behavior of the combo.

Possible values include:

  • Default
  • Do Nothing
  • Match or Add
  • Match or Closest
  • Match or First
  • Match or Nothing
  • Match or Partial
  • Match or Revert
Tag PopdownComboItem Tag name of the combo, for referencing in application code.
Title Nothing The title of the combo, usually not displayed.
Values something The various values for this combo.
Visible Values 5 The number of values visible for this combo.
Wrap False Determines whether the combo wraps the values automatically.
