An image item (ImageItem is a box item subclass that displays an image in a dialog.

An image item is essentially a box item that has its Image property set to display an image. Use the Image Editor to modify, import, or export images.

Key Properties

Property Defaults Description
Image something Specifies the image displayed by the image item.
Tag ImageItem Tag name of the image item, for referencing in application code.
Title Image The title of the image item, displayed in the center of the image item's border.
Consult the Dialog Item Key Properties table for additional properties that are inherited by this object.
