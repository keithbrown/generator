A notice dialog is a dialog subclass that communicates information to a user; typically notices inform the user that an error or some irrevocable action has occurred. Notice dialogs are usually modal; the user must respond to the notice before proceeding in an application. The display of notice dialogs vary, depending on look-and-feel.

Two types of notice dialogs are available:

  • OK Notice
    An OK notice contains the following:
    • Text for the notice
    • OK button
    • Cancel button (optional)

  • Caution Notice
    A caution notice contains the following:
    • Text for the notice
    • Yes button
    • No button
    • Cancel button (optional)

Use the Notice Editor to build notice dialogs for your projects.

For more information on notice dialogs, refer to the following:

  • Notice chapter in the Programmer's Guide.
  • NoticeDialog class in the Reference Manual.

Key Properties

Property Defaults Description
Has Cancel False Specifies whether the notice dialog has a Cancel button.
Notice Type OK Specifies the type of notice dialog, either "OK" or "Caution."
Tag NoticeDialog Tag name of the notice dialog, for referencing in application code.
Title Notice Dialog The title of the notice dialog. On most platforms, the title is displayed in the title bar of the notice dialog.
Consult the Dialog Key Properties table for additional properties that are inherited by the notice dialog.
