A dialog is an application window that provides for user interaction. The dialog's primary purpose is to hold, display, and manage dialog items. An application with a graphical user interface consists of one or more dialogs, each with its own group of dialog items to display and manage.

Use the Dialog Editor to build dialogs for your projects.

For more information on dialogs, refer to the following:

  • Dialog chapter in the Programmer's Guide.
  • Dialog class in the Reference Manual.

Dialog Subclasses

The following dialog subclasses are available from the Dialogs Page of the Classes Palette.
  • ConfirmDialog
  • NoticeDialog

Key Properties

Property Defaults Description
Background Nothing Background color of the dialog.
Border True Determines whether the dialog is drawn with a border or not.
Cursor Nothing Default cursor when the pointer is in the dialog.
Font Nothing Font to use for the dialog.
Foreground Nothing Foreground color of the dialog.
Iconifiable True Specifies whether the platform windowing system can minimize the dialog (represent it as an icon).
MaxH 32767 Maximum height (pixels) the dialog can be enlarged to.
Maximizable True Specifies whether the platform windowing system can maximize the dialog.
MaxW 32767 Maximum width (pixels) the dialog can be enlarged to.
Menu Nothing Specifies whether the dialog contains a pop-up (option) menu.
MinH 0 Minimum height (pixels) the dialog can be reduced to.
MinW 0 Minimum width (pixels) the dialog can be reduced to.
Modal False The dialog is the only active dialog in the application. A user must respond to or perform some action in a modal dialog to change the focus.
Motion False The dialog responds to motion events.
Movable True The dialog can be repositioned by the user.
Palette False The dialog is a palette or a toolbar.
Pinnable False Depending on the look-and-feel, enables pinning a dialog on the display.
Rect 
h (varies)
w (varies)
x  100
y  100
Default size (pixels) of a dialog, and its placement on the display. A Rect contains the following fields:
  • h is the height of the dialog
  • w is the width of the dialog
  • x is the distance, in points, from the origin
  • y is the distance, in points, from the origin

Note that the origin is the lower left corner of the display.

Resizable True Allows the dialog to be resized by the user.
Tag Dialog Tag name of the dialog, for referencing in application code.
Title Dialog The title of the dialog. On most platforms, the title is displayed in the title bar of the dialog.
Unclosable False The dialog cannot be closed from the platform windowing system.
