Use the Dialogs page to add a dialog (or dialog subclass) to a project.

To add a dialog to a project, the Object Editor must be displayed. In the Dialogs Palette of the Object Palette, click and drag an icon into the Application Object List.

The following dialogs are available from the Dialogs Palette of the Object Palette:

  • ConfirmDialog
  • Dialog
  • NoticeDialog
