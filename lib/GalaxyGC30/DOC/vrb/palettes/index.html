The Object Palette is located in the Object Editor. The Object Palette provides classes readily available to drop into an object's editor. Each page of the palette contains a palette of related classes; the classes are placed alphabetically on the palette. The following screenshot shows the Object Palette displaying the Dialog Item Classes.

The following pages are available from the Object palette:

  • Dialog
  • Dialog Items
  • General
  • Menu
  • OLE
  • Resource
  • Resource Arrays
Note: The OLE Palette is only available on those platforms that support OLE and ActiveX.
