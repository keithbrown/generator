Use the Menus page to add a menu or menu elements to a project in the Object Editor, or to add menu elements to a menu in the menu editor or Menubar Editor.

To add a menu to a project, the Object Editor must be displayed. In the Menu page of the Object Palette, click and drag an icon into the Application Object List.

The following objects are available on the Menu Palette:

  • Menu
  • MenuElement
  • SeparatorMenuElement
  • SubmenuMenuElement
  • ToggleMenuElement
