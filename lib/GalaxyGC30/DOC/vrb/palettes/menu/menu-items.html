Galaxy provides a variety of menu elements that can be added to a menu. Use the menu editor to add menu elements to a menu. The following are available:

  • MenuElement
    Invokes a specific command, internally represented by the assigned Tag property.

  • SeparatorMenuElement
    Draws a line on a menu. Use menu separators to visually group menu elements.

  • SubmenuMenuElement
    Invokes a submenu. Set the Menu property for the submenu element to Custom Menu to build the submenu.

  • ToggleMenuElement
    Provides a check box (toggle) that allows a user to turn on or turn off a menu element.

Key Properties

Property Defaults Description
Accelerator Nothing Defines an accelerator key for the menu element. Type the key sequence you want to use as an accelerator in the menu (for example, Ctrl+K, or ^K). Accelerator keys must be unique among all menu options.
Mnemonic Nothing Defines a mnemonic key for selecting the menu element from the keyboard. This is a one character field. The character entered must match a character in the title name. All mnemonic keys defined for menu elements in a set of menu elements must be unique.
Menu No Menu Specifies a menu attached to a submenu element.
On False For a toggle menu element, specifies that the toggle is selected.
Tag Nothing Tag name of the element, for referencing in application code.
Title Varies with each menu element. Specifies the name displayed for the menu element.
