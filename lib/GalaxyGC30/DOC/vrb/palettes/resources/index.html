Resource chapter
vresource Prefix
Use the Resources palette to add fundamental resources to a project.

The following fundamental resources are available from the Resources Palette of the Object Palette:

  • Boolean
  • Character
  • Dictionary
  • Fixed
  • Float
  • Integer
  • Opaque
  • String
  • Tag

Use the Resource Arrays palette to add fundamental resource arrays to a project.

The following fundamental resource arrays are available from the Resource Arrays Palette:

  • Boolean Array
  • Character Array
  • Compound Array
  • Fixed Array
  • Float Array
  • Integer Array
  • String Array
  • Tag Array
