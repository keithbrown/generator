vcursor Prefix






A cursor is a Galaxy specific object that can be used in your application to display graphical information. The cursor is created using the Image Editor.

A cursor is an image sixteen pixels wide and sixteen high. Using the Cursor class, you can assign the Cursor to different parts of your GUI application depending on the location of the pointer or state of the application.

A cursor is also assigned a "Hot Spot," a single pixel that specifies the position of the cursor on the screen when the pointer is clicked.

Key Properties

Property Defaults Description
Hot Spot X 7 The X coordinate of the hot spot, counting the number of pixels from the left side to the right.
Hot Spot Y 7 The Y coordinate of the hot spot, counting the number of pixels from the bottom to the top.
Tag Nothing Tag name of the cursor, for referencing in application code.
