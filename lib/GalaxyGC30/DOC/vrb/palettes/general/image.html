vimage Prefix






An Image is a Galaxy specific object that can be used in your application to display graphical information. The Image is created using the Image Editor.

Key Properties

Property Defaults Description
Dither True Specifies whether or not the image should be dithered, a technique used to create additional colors and shades by interspersing different colors.
Tag Nothing Tag name of the image, for referencing in application code.
